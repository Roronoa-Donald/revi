Chapitre 5

Calcul du CRC

Du polynome generateur au message transmis.

Objectifs

  • Comprendre le principe du CRC (Cyclic Redundancy Check).
  • Calculer un CRC a partir d'un polynome generateur.
  • Verifier si un message recu est correct.

1) Principe

On represente le message M comme un polynome M(x). On multiplie par x^k (decalage de k bits), puis on divise par G(x). Le reste R(x) est le CRC.

  • k = degre du polynome generateur G(x).
  • CRC = reste de la division polynomiale (mod 2).
  • Message transmis = M suivi des k bits du CRC.

2) Exemple rapide

G(x) = x^3 + x + 1 (1011), M = 1101.

  • Decalage: 1101 000.
  • Division mod 2 par 1011.
  • CRC = 001, message final = 1101001.

3) Exercices pratiques (5)

Chaque exercice contient au moins 3 questions. Faites le calcul a la main, puis comparez avec la correction.

Exercice 1

  • G(x) = x^3 + x + 1, M = 1101.
  • Q1: Ecrire M(x).
  • Q2: Calculer P(x) = M(x) * x^3 (decalage).
  • Q3: Trouver le CRC et le message transmis.
Correction
  • M(x) = x^3 + x^2 + 1.
  • P = 1101000.
  • CRC = 001, message transmis = 1101001.

Exercice 2

  • G(x) = x^4 + x^2 + x, M = 111101101.
  • Q1: Ecrire M(x).
  • Q2: Calculer le CRC.
  • Q3: Le message recu 1111011010100 est-il correct ?
Correction
  • M(x) = x^8 + x^7 + x^6 + x^5 + x^3 + x^2 + 1.
  • CRC = 0100.
  • Message transmis = 1111011010100, donc le message recu est correct.

Exercice 3

  • G(x) = x^3 + x^2 + 1, M = 101100.
  • Q1: Ecrire M(x).
  • Q2: Trouver le CRC.
  • Q3: Donner le message transmis.
Correction
  • M(x) = x^5 + x^3 + x^2.
  • CRC = 111.
  • Message transmis = 101100111.

Exercice 4

  • G(x) = x^5 + x^2 + 1, M = 1001110.
  • Q1: Donner k et le decalage.
  • Q2: Calculer le CRC.
  • Q3: Donner le message transmis.
Correction
  • k = 5, decalage de 5 bits.
  • CRC = 10100.
  • Message transmis = 100111010100.

Exercice 5

  • G(x) = x^2 + x + 1, M = 110101.
  • Q1: Ecrire M(x).
  • Q2: Calculer le CRC.
  • Q3: Donner le message transmis.
Correction
  • M(x) = x^5 + x^4 + x^2 + 1.
  • CRC = 10.
  • Message transmis = 11010110.
