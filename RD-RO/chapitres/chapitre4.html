Module 04

Le Dress Code du Mathématicien

L'algorithme du Simplexe, c'est un club très sélect. Il refuse les inégalités. Pour entrer, votre modèle doit enfiler un costume strict : la "Forme Standard".

1. Les 4 Catégories d'Éléments d'un Programme Linéaire

Avant de transformer un modèle en forme standard, il faut bien identifier les 4 types d'éléments qui composent tout programme linéaire. Le prof Dr. TCHAZINO les classe ainsi :

1. Variables (ou Activités) — $x_j$

Ce sont les variables de décision : les quantités que l'on cherche à déterminer.

Ex : $x_1$ = nombre de bureaux luxe, $x_2$ = nombre de bureaux standard.

2. Coefficients Économiques — $c_j$

La contribution unitaire de chaque activité $x_j$ à la fonction objectif $Z$.

Ex : $c_1 = 5$ (chaque bureau luxe rapporte 5 milliers de F).

3. Ressources (ou Disponibilités) — $b_i$

Les limitations disponibles pour chaque contrainte (heures machine, matière première, budget…).

Ex : $b_1 = 2\,000$ heures de découpage disponibles.

4. Coefficients Techniques — $a_{ij}$

La quantité de ressource $i$ consommée par une unité de l'activité $j$.

Ex : $a_{12} = 4$ signifie qu'un bureau standard consomme 4h de découpage.

Écriture Générale d'un PL

$$\text{Max (ou Min) } Z = \sum_{j=1}^{n} c_j \cdot x_j = c_1 x_1 + c_2 x_2 + \ldots + c_n x_n$$

Sous les contraintes :

$$\sum_{j=1}^{n} a_{ij} \cdot x_j \leq b_i \quad \text{pour } i = 1, 2, \ldots, m$$

$$x_j \geq 0 \quad \text{pour } j = 1, 2, \ldots, n$$

Où $n$ = nombre de variables (activités) et $m$ = nombre de contraintes (ressources).

2. De la Forme Canonique à la Forme Standard

L'algorithme du Simplexe exige la forme standard : toutes les contraintes doivent être des égalités ($=$), pas des inégalités ($\leq$ ou $\geq$). Voici comment convertir chaque type de contrainte.

≤ 2.1 Contrainte "Plus Petit Que" → Variable d'Écart ($e_i$)

On ajoute une variable d'écart (slack variable) $e_i \geq 0$ qui représente le stock inutilisé de la ressource $i$.

// Forme Canonique
$2x_1 + 4x_2 \leq 2\,000$
// Forme Standard
$2x_1 + 4x_2 + \textcolor{cyan}{e_1} = 2\,000$

$e_1$ = heures de découpage non utilisées. Si $e_1 = 0$ → la ressource est entièrement consommée (contrainte saturée).

≥ 2.2 Contrainte "Plus Grand Que" → Variable de Surplus ($s_i$)

On soustrait une variable de surplus (surplus variable) $s_i \geq 0$ qui représente le dépassement au-delà du minimum exigé.

// Forme Canonique
$x_1 + x_2 \geq 4$
// Forme Standard
$x_1 + x_2 - \textcolor{orange}{s_1} = 4$

⚠️ Attention : soustraire $s_i$ ne fournit pas de base initiale pour le Simplexe. Il faudra ajouter une variable artificielle (voir Big-M et Deux Phases au chapitre 7).

= 2.3 Contrainte d'Égalité → Variable Artificielle ($a_i$)

Si la contrainte est déjà une égalité, on ne peut pas ajouter d'écart. On introduit une variable artificielle $a_i$ temporaire pour initialiser le Simplexe, puis on la pénalise pour la forcer à zéro.

// Contrainte d'égalité
$x_1 + 2x_2 = 6$
// Avec artificielle
$x_1 + 2x_2 + \textcolor{violet}{a_1} = 6$

Résumé : Conversion vers la Forme Standard

Type de contrainte Action Variable ajoutée Signe dans l'équation
$\leq$ Ajouter variable d'écart $+ e_i$ $+$
$\geq$ Soustraire variable de surplus + artificielle $- s_i + a_i$ $-$ et $+$
$=$ Ajouter variable artificielle $+ a_i$ $+$

3. Compter les Variables d'Écart (Question d'Examen ⚠️)

Question type du Prof (Examen Final, Q6)

"Un programme linéaire de 5 variables de décision et 3 contraintes fonctionnelles possède combien de variables d'écart ?"

Réponse : On ajoute une variable d'écart (ou de surplus) par contrainte fonctionnelle. Donc : 3 variables d'écart.

Règle Générale

Avec $n$ variables de décision et $m$ contraintes fonctionnelles :

$$\text{Nombre de variables d'écart} = m$$

$$\text{Nombre total de variables (forme standard)} = n + m$$

Exemple : $n = 5$ variables, $m = 3$ contraintes → $3$ variables d'écart → $5 + 3 = 8$ variables en forme standard.

4. Exemple Complet — L'Ébéniste (Problème du Prof)

Un ébéniste fabrique deux types de bureaux : bureau luxe ($x_1$) et bureau standard ($x_2$). Voici les données :

Opération Bureau Luxe ($x_1$) Bureau Standard ($x_2$) Disponibilité ($b_i$)
Découpage $a_{11} = 2$ $a_{12} = 4$ 2 000 h
Ponçage $a_{21} = 5$ $a_{22} = 2$ 2 500 h
Assemblage $a_{31} = 4$ $a_{32} = 7$ 3 500 h
Finition $a_{41} = 3$ $a_{42} = 1$ 1 500 h
Bénéfice ($c_j$) $c_1 = 5$ (milliers F) $c_2 = 4$ (milliers F)
Modèle (Forme Canonique)

$$\text{Max } Z = 5x_1 + 4x_2$$

$2x_1 + 4x_2 \leq 2\,000$

$5x_1 + 2x_2 \leq 2\,500$

$4x_1 + 7x_2 \leq 3\,500$

$3x_1 + x_2 \leq 1\,500$

$x_1, x_2 \geq 0$

Forme Standard (+ 4 variables d'écart)

$$\text{Max } Z = 5x_1 + 4x_2 + 0e_1 + 0e_2 + 0e_3 + 0e_4$$

$2x_1 + 4x_2 + e_1 = 2\,000$

$5x_1 + 2x_2 + e_2 = 2\,500$

$4x_1 + 7x_2 + e_3 = 3\,500$

$3x_1 + x_2 + e_4 = 1\,500$

$x_1, x_2, e_1, e_2, e_3, e_4 \geq 0$

Identification des 4 catégories :
• Variables/Activités : $x_1, x_2$ (les 2 types de bureaux)
• Coefficients économiques : $c_1 = 5$, $c_2 = 4$ (bénéfices unitaires)
• Ressources : $b_1 = 2\,000$, $b_2 = 2\,500$, $b_3 = 3\,500$, $b_4 = 1\,500$ (heures dispos)
• Coefficients techniques : $a_{11} = 2$, $a_{12} = 4$, $a_{21} = 5$, … (conso par unité)
