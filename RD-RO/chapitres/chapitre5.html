Module 05

Les Types de Solutions

Dans l'univers de l'optimisation, toutes les solutions ne se valent pas. Certaines sont impossibles (illégales), d'autres sont possibles mais médiocres, et une seule est "optimale".

1. Vocabulaire Essentiel (Terminologie du Prof)

Solution Quelconque

N'importe quel vecteur $(x_1, x_2, \ldots, x_n)$ sans condition. Il peut violer des contraintes ou être négatif. C'est un point "au hasard".

Pas forcément valide.

Solution Réalisable (= Solution Admissible)

Tout point qui respecte TOUTES les contraintes (y compris les contraintes de non-négativité $x_j \geq 0$).

⚠️ Terme courant : En examen, on utilise indifféremment "solution admissible" et "solution réalisable". C'est la même chose.

C'est un "candidat légal".

Solution de Base

Un point d'intersection des contraintes. En géométrie, c'est un SOMMET (un coin) du polyèdre.

Définition Formelle (Examen)

Pour un PL avec $n$ variables de décision et $m$ contraintes fonctionnelles (en forme standard : $n + m$ variables au total) :

Une solution de base est obtenue en fixant au moins $n$ variables à zéro (variables hors la base) et en résolvant le système pour les $m$ variables restantes (variables dans la base).

Nombre de solutions de base possibles = $C_{n+m}^{m} = \frac{(n+m)!}{m! \cdot n!}$

L'optimum est toujours en un sommet !

Solution Optimale

La solution réalisable de base qui donne la meilleure valeur de $Z$ (max ou min selon l'objectif).

2. Variables Dans la Base (VDB) vs Hors la Base (VHB)

Ce vocabulaire est essentiel pour l'examen. À chaque itération du Simplexe, on distingue deux ensembles de variables :

Variables Dans la Base (VDB)

  • • Ce sont les variables qui ont une valeur non nulle (en général)
  • • Il y en a exactement $m$ (autant que de contraintes)
  • • Elles forment une matrice identité dans le tableau
  • • Leur valeur se lit dans la colonne RHS
Au départ : VDB = {$e_1, e_2, \ldots, e_m$}

Variables Hors la Base (VHB)

  • • Ce sont les variables fixées à zéro
  • • Il y en a exactement $n$ (les variables de décision au départ)
  • • Elles ne contribuent pas au profit actuellement
  • • C'est parmi elles qu'on choisit la variable entrante
Au départ : VHB = {$x_1, x_2, \ldots, x_n$}

Question d'Examen Typique (Q4)

"Si un PL a 5 variables de décision et 3 contraintes fonctionnelles, que peut-on dire d'une variable hors la base ?"

Réponse : Une variable hors la base a une valeur nulle (= 0) dans la solution courante. Elle ne "participe" pas encore à la production.

3. Le Test d'Optimalité

Définition (Question d'Examen Q5)

"Qu'appelle-t-on test d'optimalité dans la méthode du Simplexe ?"

Réponse : Le test d'optimalité consiste à examiner les coefficients de la ligne $Z$ (ligne objectif) pour les variables hors la base :

Cas Maximisation

La solution est optimale si et seulement si :

$$\text{Tous les } \bar{c}_j \geq 0 \text{ dans la ligne Z}$$

(Aucun coefficient négatif → plus d'amélioration possible)

Cas Minimisation

La solution est optimale si et seulement si :

$$\text{Tous les } \bar{c}_j \leq 0 \text{ dans la ligne Z}$$

(Aucun coefficient positif → plus de réduction possible)

Que signifient ces coefficients ?

Les coefficients $\bar{c}_j$ dans la ligne Z des variables hors la base s'appellent les coûts réduits (ou gains marginaux). Ils indiquent de combien $Z$ changerait si on introduisait une unité de cette variable dans la base.

• $\bar{c}_j < 0$ en maximisation → produire $x_j$ améliorerait Z de $|\bar{c}_j|$ par unité

• $\bar{c}_j \geq 0$ en maximisation → produire $x_j$ n'améliorerait pas Z

4. Le Théorème Fondamental de la PL

Conséquence : Au lieu de tester une infinité de points, le Simplexe parcourt uniquement les sommets du polyèdre, en passant d'un sommet à un voisin meilleur, jusqu'à atteindre l'optimum.

5. Récapitulatif pour l'Examen

Terme Définition Courte Exemple
Solution admissible = Solution réalisable : respecte toutes les contraintes $(3, 2)$ avec $3 + 2 \leq 10$ ✓
Variable dans la base (VDB) Variable de valeur non nulle dans la solution courante $e_1 = 20$ → VDB
Variable hors la base (VHB) Variable fixée à 0, ne participe pas à la solution $x_1 = 0$ → VHB
Solution de base $n$ variables = 0 (VHB), $m$ variables résolues (VDB) Sommet du polyèdre
Test d'optimalité Vérifier si tous les $\bar{c}_j \geq 0$ (max) dans la ligne Z Oui → STOP, sinon → continuer
Gain marginal Amélioration de Z par unité de variable entrante $\bar{c}_j = -5$ → gain de 5 par unité

🧠 MÉMO TECHNIQUE — Types de Solutions

« O.M.I.N — les 4 cas possibles »

  • Optimale unique → un seul sommet optimal
  • Multiple → plusieurs sommets donnent le même Z optimal (arête entière)
  • Infaisable → aucune solution ne respecte toutes les contraintes
  • Non-bornée → Z peut croître à l'infini (problème mal posé)

Si les contraintes se contredisent → Infaisable. Si le domaine n'est pas fermé → Non-borné.
