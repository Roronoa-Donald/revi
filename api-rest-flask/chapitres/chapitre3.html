Chapitre 3

Flask minimal

Coder une API Python claire.

Installer et demarrer

Flask est un micro-framework Python. Il permet de declarer rapidement des routes HTTP.

jsonify transforme des donnees Python en reponse JSON.

pip install flask
python app.py
Memo rapide : Flask = routes simples + fonctions Python.

Premiere route

Le decorateur @app.get indique la methode et la route.

La fonction retourne les donnees de la reponse.

from flask import Flask, jsonify

app = Flask(__name__)

@app.get("/students")
def list_students():
    return jsonify([])

app.run(debug=True)
Tip : @app.get("/route") se lit: quand GET arrive ici, execute cette fonction.

Body JSON

Pour POST ou PUT, le client envoie souvent un body JSON. Flask le lit avec request.get_json().

Il faut verifier les donnees avant de les accepter.

from flask import request

data = request.get_json() or {}
name = data.get("name")
Piege : Ne fais pas confiance au body. Une API doit valider les champs obligatoires.

Atelier guide

Ecris une mini route Flask GET /students.

Correction guidee :
from flask import Flask, jsonify

app = Flask(__name__)

@app.get("/students")
def list_students():
    return jsonify([{"id": 1, "name": "Lisa"}])

La route lit une collection students et renvoie une liste JSON.

Questions de fin de chapitre

Reponds avant de lire la correction. Elle se debloque apres une bonne reponse, ou apres deux erreurs.

1. Qu'est-ce que Flask dans l'écosystème Python ?
Correction : Flask est un framework léger en Python permettant de déclarer facilement des routes HTTP et de servir des réponses.
2. Quelle commande permet d'installer Flask dans votre environnement de développement ?
Correction : En Python, l'installateur standard de paquets est pip. On installe Flask via `pip install flask`.
3. Comment déclare-t-on l'instance principale de l'application Flask ?
Correction : On initialise l'application en instanciant `Flask` avec `__name__` pour que le framework localise les ressources.
4. Quel décorateur Flask associe une fonction à des requêtes GET sur la route '/status' ?
Correction : Flask moderne propose `@app.get('/route')` comme raccourci explicite pour associer la méthode GET.
5. À quoi sert la fonction 'jsonify' fournie par Flask ?
Correction : jsonify convertit les données en chaîne JSON et configure automatiquement l'en-tête de réponse sur `application/json`.
