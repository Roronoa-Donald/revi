REST rapide

  • GET /students: liste.
  • GET /students/42: detail.
  • POST /students: creation.
  • PUT /students/42: mise a jour.
  • DELETE /students/42: suppression.

Codes utiles

  • 200 OK.
  • 201 Created.
  • 400 donnees invalides.
  • 401 non authentifie.
  • 403 interdit.
  • 404 introuvable.
  • 500 erreur serveur.
