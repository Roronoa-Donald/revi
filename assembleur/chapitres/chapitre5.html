Chapitre 5

Fondements de l'assembleur AVR

Registres, structure de programme et instructions essentielles.

Objectifs

  • Identifier la structure d'un programme AVR.
  • Utiliser les registres R0-R31 et le status register.
  • Comprendre LDI, MOV, LDS, STS et les branches.

1) Structure minimale

Un programme AVR contient des directives, un point d'entree et des labels.

Exemple: .include, .org, puis le label start: pour le code.
  • Initialiser la pile si besoin.
  • Configurer les ports E/S.
  • Boucle principale avec RJMP.

2) Registres et status register

Les 32 registres generaux (R0-R31) sont utilises pour les calculs rapides. Le status register (SREG) contient les flags.

3) Instructions de base

  • LDI: charger une constante dans un registre.
  • MOV: copier un registre vers un autre.
  • LDS/STS: lire/ ecrire en memoire.
  • IN/OUT: acces aux registres E/S.
  • RJMP/BRNE: branchements.

Exemple commenté

  • LDI R16, 0xFF
  • OUT DDRB, R16 (PORTB en sortie)
  • LDI R16, 0x01
  • OUT PORTB, R16 (allumer LED PB0)

Recap / Cheat Sheet

Directives:
  • .include
  • .org
  • labels
Instructions:
  • LDI / MOV
  • LDS / STS
  • RJMP / BRNE
E/S:
  • DDR = direction
  • PORT = sortie
  • PIN = lecture
