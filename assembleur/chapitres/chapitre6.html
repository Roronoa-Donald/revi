Chapitre 6

NASM x86_64: bases

Sections, registres et adressage memoire en assembleur x64.

Objectifs

  • Comprendre .data, .bss, .text.
  • Connaitre les registres 64 bits usuels.
  • Utiliser MOV et LEA correctement.

1) Sections NASM

  • .data: donnees initialisees.
  • .bss: donnees non initialisees.
  • .text: code machine.
En examen, on verifie souvent la structure .data/.bss/.text.

2) Registres x86_64

Les registres generaux: RAX, RBX, RCX, RDX, RSI, RDI, RBP, RSP, R8-R15.

  • RAX: accumulateur, souvent retour de fonctions.
  • RSP/RBP: pile (stack).
  • RIP: pointeur d'instruction.

3) MOV vs LEA

MOV copie une valeur. LEA calcule une adresse sans dereferencer.

  • MOV RAX, [var] lit la valeur en memoire.
  • LEA RAX, [var] place l'adresse de var.

4) Adressage memoire

On peut utiliser une adresse simple ou une formule base + index * scale + offset.

  • [RBX]
  • [RBX + 8]
  • [RBX + RCX * 4]

Recap / Cheat Sheet

Sections:
  • .data: init
  • .bss: reserve
  • .text: code
Registres:
  • RAX/RBX/RCX/RDX
  • RSI/RDI
  • RSP/RBP/RIP
Adressage:
  • [base]
  • [base+offset]
  • [base+index*scale]
