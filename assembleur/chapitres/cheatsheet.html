Cheat Sheet

Resume global - Assembleur

A utiliser pour la revision rapide avant controle ou examen.

Chapitre 1 - Numeration & logique

  • Base 2/8/10/16, lecture positionnelle (poids 2^n).
  • Complement a 2: inversion + 1, plage 8 bits = -128..127.
  • Hexa = groupes de 4 bits, octal = groupes de 3 bits.
  • AND garder, OR forcer, XOR basculer.

Chapitre 2 - CPU et cycle

  • Cycle: fetch, decode, execute, write-back.
  • Registres > cache > RAM > disque (du plus rapide au plus lent).
  • Flags: Z (zero), C (carry), V (overflow), N (negatif).
  • Interruptions: detour vers une ISR, puis retour.

Chapitre 3 - AVR

  • Architecture Harvard: Flash (code) + SRAM (data).
  • EEPROM = donnees persistantes.
  • Ports: DDR (direction), PORT (ecriture/pull-up), PIN (lecture).
  • Timers + prescaler = base des interruptions periodiques.

Chapitre 4 - Toolchain

  • Source -> .o -> .elf -> .hex -> flash.
  • avr-gcc, avr-objcopy, avrdude.
  • Verifier: MCU + programmeur + port COM.
  • LED blink: DDR sortie, PORT on/off, boucle delai.

Chapitre 5 - AVR assembleur

  • LDI (R16-R31), MOV, ADD/SUB, CP/CPI.
  • IN/OUT = espace E/S, LDS/STS = SRAM.
  • RJMP/BREQ/BRNE pour les boucles.
  • Pointeurs: X=R27:R26, Y=R29:R28, Z=R31:R30.

Chapitre 6 - NASM x86_64

  • .data / .bss / .text, db/dw/dd/dq.
  • RAX/EAX/AX/AL = meme registre a 4 tailles.
  • [ ] = memoire, sans [ ] = registre/valeur.
  • MOV lit/ecrit, LEA calcule une adresse.

Chapitre 7 - NASM avance

  • Pile: CALL empile retour, RET depile, alignement 16.
  • Syscall Linux: RAX numero, RDI/RSI/RDX/R10/R8/R9 args.
  • ABI Windows: RCX/RDX/R8/R9 pour les args.
  • Build Windows: nasm -f win64 + gcc.

Memos techniques (revision rapide)

  • AND garder, OR forcer, XOR basculer.
  • Flash = code, SRAM = variables, EEPROM = persistant.
  • RSP = pile, RIP = instruction.
  • LDI seulement R16-R31.
