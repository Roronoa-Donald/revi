Dans ce premier chapitre, nous allons découvrir le langage PHP et ses fondamentaux avant d'aborder la programmation orientée objet.

4 Les fonctions en PHP

Les fonctions permettent de réutiliser du code et de structurer votre programme. Une fonction est un bloc de code qui peut être appelé plusieurs fois.

Déclaration et utilisation

fonctions.php 
<?php
// Fonction simple
function direBonjour() {
    echo "Bonjour !<br>";
}

// Appel de la fonction
direBonjour();

// Fonction avec paramètres
function saluer($nom, $titre = "M.") {
    echo "Bonjour $titre $nom !<br>";
}

saluer("Dupont"); // Utilise la valeur par défaut
saluer("Martin", "Mme"); // Remplace la valeur par défaut

// Fonction avec retour de valeur
function addition($a, $b) {
    return $a + $b;
}

$resultat = addition(5, 3);
echo "5 + 3 = $resultat<br>";

// Fonction avec plusieurs valeurs de retour
function calculer($a, $b) {
    $somme = $a + $b;
    $produit = $a * $b;
    $difference = $a - $b;
    
    return array($somme, $produit, $difference);
}

list($s, $p, $d) = calculer(10, 5);
echo "<br>Somme : $s<br>";
echo "Produit : $p<br>";
echo "Différence : $d<br>";

// Passage par référence
function incrementer(&$nombre) {
    $nombre++;
}

$x = 10;
echo "<br>Avant : $x<br>";
incrementer($x);
echo "Après : $x<br>";

// Fonction avec nombre variable d'arguments
function somme(...$nombres) {
    $total = 0;
    foreach ($nombres as $nombre) {
        $total += $nombre;
    }
    return $total;
}

echo "<br>Somme : " . somme(1, 2, 3, 4, 5);
?>
Résultat

Bonnes pratiques

  • Donnez des noms explicites à vos fonctions
  • Une fonction doit avoir une seule responsabilité
  • Documentez vos fonctions complexes
  • Utilisez des valeurs par défaut pour les paramètres optionnels

5 Introduction à la POO

Qu'est-ce que la POO ?

La Programmation Orientée Objet est un paradigme de programmation qui organise le code autour d'objets plutôt que de fonctions et de données.

Les 4 piliers de la POO

Encapsulation

Regrouper les données et les méthodes qui les manipulent dans une seule entité (classe).

Exemple : Une classe Voiture contient les propriétés (couleur, marque) et les méthodes (démarrer, arrêter).

Héritage

Créer de nouvelles classes à partir de classes existantes, en héritant de leurs propriétés et méthodes.

Exemple : Une classe VoitureSport hérite de Voiture et ajoute des fonctionnalités spécifiques.

Polymorphisme

Utiliser une même interface pour différentes formes d'objets.

Exemple : Plusieurs classes peuvent avoir une méthode afficher() qui fonctionne différemment.

Abstraction

Masquer les détails complexes et exposer seulement les fonctionnalités essentielles.

Exemple : Vous utilisez une voiture sans connaître le fonctionnement du moteur.

Pourquoi utiliser la POO ?

Réutilisabilité du code

Les classes peuvent être réutilisées dans différents projets

Maintenance facilitée

Le code est mieux organisé et plus facile à modifier

Code plus lisible

Structure claire et logique du programme

Travail en équipe

Plusieurs développeurs peuvent travailler sur différentes classes

6 Classes et objets en PHP

Créer une classe

Une classe est un modèle qui définit la structure et le comportement des objets. Un objet est une instance de cette classe.

classe-simple.php 
<?php
// Déclaration d'une classe
class Voiture {
    // Propriétés (attributs)
    public $marque;
    public $couleur;
    public $vitesseMax;
    
    // Méthode
    public function demarrer() {
        echo "La voiture démarre !<br>";
    }
    
    public function afficherInfo() {
        echo "Marque : {$this->marque}<br>";
        echo "Couleur : {$this->couleur}<br>";
        echo "Vitesse max : {$this->vitesseMax} km/h<br>";
    }
}

// Créer un objet (instanciation)
$maVoiture = new Voiture();

// Définir les propriétés
$maVoiture->marque = "Peugeot";
$maVoiture->couleur = "Rouge";
$maVoiture->vitesseMax = 180;

// Appeler les méthodes
$maVoiture->afficherInfo();
$maVoiture->demarrer();

// Créer un autre objet
echo "<br>";
$autreVoiture = new Voiture();
$autreVoiture->marque = "Renault";
$autreVoiture->couleur = "Bleue";
$autreVoiture->vitesseMax = 200;
$autreVoiture->afficherInfo();
?>
Résultat

Explication

  • class Voiture : Déclare la classe
  • public $marque : Propriété publique
  • $this : Fait référence à l'objet courant
  • new Voiture() : Crée une instance de la classe
  • $maVoiture->marque : Accède à une propriété
  • $maVoiture->demarrer() : Appelle une méthode

Constructeur et Destructeur

constructeur.php 
<?php
class Personne {
    private $nom;
    private $age;
    
    // Constructeur : appelé automatiquement à la création
    public function __construct($nom, $age) {
        $this->nom = $nom;
        $this->age = $age;
        echo "Création de l'objet : $nom<br>";
    }
    
    // Destructeur : appelé à la destruction de l'objet
    public function __destruct() {
        echo "Destruction de l'objet : {$this->nom}<br>";
    }
    
    public function sePresenter() {
        echo "Je m'appelle {$this->nom} et j'ai {$this->age} ans.<br>";
    }
    
    // Getters
    public function getNom() {
        return $this->nom;
    }
    
    public function getAge() {
        return $this->age;
    }
    
    // Setters
    public function setNom($nom) {
        $this->nom = $nom;
    }
    
    public function setAge($age) {
        if ($age > 0 && $age < 150) {
            $this->age = $age;
        }
    }
}

// Utilisation
$personne1 = new Personne("Alice", 25);
$personne1->sePresenter();

echo "<br>";

$personne2 = new Personne("Bob", 30);
$personne2->sePresenter();

// Modification via setter
$personne2->setAge(31);
echo "Nouvel âge de Bob : " . $personne2->getAge() . " ans<br>";
?>

<?php
class CompteBancaire {
    public $titulaire;        // Accessible partout
    protected $solde;         // Accessible dans la classe et ses enfants
    private $numeroCompte;    // Accessible uniquement dans cette classe
    
    public function __construct($titulaire, $solde, $numeroCompte) {
        $this->titulaire = $titulaire;
        $this->solde = $solde;
        $this->numeroCompte = $numeroCompte;
    }
    
    // Méthode publique pour accéder au solde
    public function getSolde() {
        return $this->solde;
    }
    
    public function deposer($montant) {
        if ($montant > 0) {
            $this->solde += $montant;
            echo "Dépôt de $montant € effectué.<br>";
        }
    }
    
    public function retirer($montant) {
        if ($montant > 0 && $montant <= $this->solde) {
            $this->solde -= $montant;
            echo "Retrait de $montant € effectué.<br>";
        } else {
            echo "Solde insuffisant !<br>";
        }
    }
    
    public function afficherInfos() {
        echo "Titulaire : {$this->titulaire}<br>";
        echo "Solde : {$this->solde} €<br>";
        echo "Numéro : {$this->numeroCompte}<br>";
    }
}

// Utilisation
$compte = new CompteBancaire("Martin", 1000, "FR123456");

echo $compte->titulaire; // OK - public
echo "<br>";

// echo $compte->solde; // ERREUR - protected
// echo $compte->numeroCompte; // ERREUR - private

// On utilise les méthodes publiques
echo "Solde : " . $compte->getSolde() . " €<br>";
$compte->deposer(500);
$compte->retirer(200);
echo "<br>";
$compte->afficherInfos();
?>

Bonne pratique

Utilisez private pour vos propriétés et créez des méthodes publiques (getters/setters) pour y accéder. Cela protège vos données et permet de contrôler les modifications.

Exercice 1 Facile

Créer une classe Produit

Créez une classe Produit avec les propriétés suivantes :

  • nom (string)
  • prix (float)
  • quantite (int)

Ajoutez :

  • Un constructeur pour initialiser ces propriétés
  • Une méthode afficherProduit() qui affiche les informations
  • Une méthode calculerTotal() qui retourne le prix total (prix × quantité)

Créez ensuite 2 objets Produit et affichez leurs informations.

Solution 
<?php
class Produit {
    private $nom;
    private $prix;
    private $quantite;
    
    public function __construct($nom, $prix, $quantite) {
        $this->nom = $nom;
        $this->prix = $prix;
        $this->quantite = $quantite;
    }
    
    public function afficherProduit() {
        echo "Produit : {$this->nom}<br>";
        echo "Prix unitaire : {$this->prix} €<br>";
        echo "Quantité : {$this->quantite}<br>";
        echo "Total : " . $this->calculerTotal() . " €<br>";
        echo "<br>";
    }
    
    public function calculerTotal() {
        return $this->prix * $this->quantite;
    }
}

// Utilisation
$produit1 = new Produit("Ordinateur", 899.99, 2);
$produit1->afficherProduit();

$produit2 = new Produit("Souris", 19.99, 5);
$produit2->afficherProduit();
?>
Explication
  • Les propriétés sont private pour l'encapsulation
  • Le constructeur initialise les valeurs à la création
  • Les méthodes publiques permettent d'interagir avec l'objet
  • $this fait référence à l'instance courante
Exercice 2 Moyen

Système de gestion d'étudiants

Créez une classe Etudiant avec :

  • Propriétés : nom, prenom, notes (tableau)
  • Constructeur pour nom et prénom
  • Méthode ajouterNote($note) pour ajouter une note
  • Méthode calculerMoyenne() qui retourne la moyenne
  • Méthode obtenirMention() qui retourne la mention selon la moyenne

Mentions : Très Bien (≥16), Bien (≥14), Assez Bien (≥12), Passable (≥10), Insuffisant (<10)

Solution 
<?php
class Etudiant {
    private $nom;
    private $prenom;
    private $notes = [];
    
    public function __construct($nom, $prenom) {
        $this->nom = $nom;
        $this->prenom = $prenom;
    }
    
    public function ajouterNote($note) {
        if ($note >= 0 && $note <= 20) {
            $this->notes[] = $note;
            echo "Note $note ajoutée pour {$this->prenom} {$this->nom}<br>";
        } else {
            echo "Note invalide !<br>";
        }
    }
    
    public function calculerMoyenne() {
        if (empty($this->notes)) {
            return 0;
        }
        $somme = array_sum($this->notes);
        return $somme / count($this->notes);
    }
    
    public function obtenirMention() {
        $moyenne = $this->calculerMoyenne();
        
        if ($moyenne >= 16) {
            return "Très Bien";
        } elseif ($moyenne >= 14) {
            return "Bien";
        } elseif ($moyenne >= 12) {
            return "Assez Bien";
        } elseif ($moyenne >= 10) {
            return "Passable";
        } else {
            return "Insuffisant";
        }
    }
    
    public function afficherResultats() {
        echo "<br>=== Résultats de {$this->prenom} {$this->nom} ===<br>";
        echo "Notes : " . implode(", ", $this->notes) . "<br>";
        $moyenne = $this->calculerMoyenne();
        echo "Moyenne : " . number_format($moyenne, 2) . "/20<br>";
        echo "Mention : " . $this->obtenirMention() . "<br>";
    }
}

// Utilisation
$etudiant1 = new Etudiant("Martin", "Alice");
$etudiant1->ajouterNote(15);
$etudiant1->ajouterNote(16);
$etudiant1->ajouterNote(14);
$etudiant1->ajouterNote(17);
$etudiant1->afficherResultats();

$etudiant2 = new Etudiant("Dupont", "Bob");
$etudiant2->ajouterNote(12);
$etudiant2->ajouterNote(13);
$etudiant2->ajouterNote(11);
$etudiant2->afficherResultats();
?>
Explication
  • Le tableau $notes est initialisé vide
  • array_sum() calcule la somme des notes
  • count() compte le nombre d'éléments
  • Les conditions en cascade déterminent la mention
  • number_format() formate le nombre avec 2 décimales

class Livre {

        $this->titre = $titre;
        $this->auteur = $auteur;
        $this->annee = $annee;
        $this->disponible = true;
    }
    
    public function getTitre() {
        return $this->titre;
    }
    
    public function isDisponible() {
        return $this->disponible;
    }
    
    public function emprunter() {
        if ($this->disponible) {
            $this->disponible = false;
            echo "✓ '{$this->titre}' a été emprunté.<br>";
            return true;
        } else {
            echo "✗ '{$this->titre}' n'est pas disponible.<br>";
            return false;
        }
    }
    
    public function retourner() {
        if (!$this->disponible) {
            $this->disponible = true;
            echo "✓ '{$this->titre}' a été retourné.<br>";
        } else {
            echo "✗ Ce livre n'était pas emprunté.<br>";
        }
    }
    
    public function afficherInfo() {
        $statut = $this->disponible ? "Disponible" : "Emprunté";
        echo "- '{$this->titre}' par {$this->auteur} ({$this->annee}) - [$statut]<br>";
    }
}

class Bibliotheque {
    private static $livres = [];
    
    public static function ajouterLivre($livre) {
        self::$livres[] = $livre;
        echo "Livre ajouté : '{$livre->getTitre()}'<br>";
    }
    
    public static function rechercherParTitre($titre) {
        foreach (self::$livres as $livre) {
            if (stripos($livre->getTitre(), $titre) !== false) {
                return $livre;
            }
        }
        return null;
    }
    
    public static function afficherLivresDisponibles() {
        echo "<br>=== Livres disponibles ===<br>";
        $compte = 0;
        foreach (self::$livres as $livre) {
            if ($livre->isDisponible()) {
                $livre->afficherInfo();
                $compte++;
            }
        }
        if ($compte == 0) {
            echo "Aucun livre disponible.<br>";
        }
    }
    
    public static function afficherTousLesLivres() {
        echo "<br>=== Tous les livres ===<br>";
        foreach (self::$livres as $livre) {
            $livre->afficherInfo();
        }
    }
    
    public static function compterLivres() {
        $total = count(self::$livres);
        $disponibles = 0;
        foreach (self::$livres as $livre) {
            if ($livre->isDisponible()) {
                $disponibles++;
            }
        }
        echo "<br>Total de livres : $total<br>";
        echo "Disponibles : $disponibles<br>";
        echo "Empruntés : " . ($total - $disponibles) . "<br>";
    }
}

Explication
  • Encapsulation : Les propriétés sont privées avec des getters
  • Membres statiques : $livres est partagé par tous les objets
  • Gestion d'état : $disponible suit l'état du livre
  • Recherche : stripos() permet une recherche insensible à la casse
  • Validation : Les méthodes vérifient l'état avant modification
