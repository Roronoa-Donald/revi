1. Introduction à PDO

PDO (PHP Data Objects) est une extension définissant une interface pour accéder à une base de données depuis PHP. Elle permet d'utiliser le même code pour interagir avec différents SGBD (MySQL, PostgreSQL, etc.).

2. Connexion à la BDD

PHP 
try {
    $dsn = 'mysql:host=localhost;dbname=test;charset=utf8';
    $user = 'root';
    $password = '';
    
    $bdd = new PDO($dsn, $user, $password);
    
    // Activation des erreurs PDO
    $bdd->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Connexion réussie !";
} catch (Exception $e) {
    die('Erreur : ' . $e->getMessage());
}

3. Exécuter des requêtes (Query)

Pour des requêtes simples sans variables utilisateur :

PHP 
$reponse = $bdd->query('SELECT * FROM jeux_video');

while ($donnees = $reponse->fetch()) {
    echo $donnees['nom'] . "\n";
}

$reponse->closeCursor();

4. Requêtes Préparées

Pour la sécurité (éviter les injections SQL) et la performance :

PHP 
$possesseur = 'Michel';
$req = $bdd->prepare('SELECT nom, prix FROM jeux_video WHERE possesseur = ?');
$req->execute(array($possesseur));

while ($donnees = $req->fetch()) {
    echo $donnees['nom'] . ' coûte ' . $donnees['prix'] . ' euros' . "\n";
}
