1. Introduction à PDO

PDO (PHP Data Objects) est une extension définissant une interface pour accéder à une base de données depuis PHP. Elle permet d'utiliser le même code pour interagir avec différents SGBD (MySQL, PostgreSQL, etc.).

2. Connexion à la BDD

PHP 
try {
    $dsn = 'mysql:host=localhost;dbname=test;charset=utf8';
    $user = 'root';
    $password = '';
    
    $bdd = new PDO($dsn, $user, $password);
    
    // Activation des erreurs PDO
    $bdd->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Connexion réussie !";
} catch (Exception $e) {
    die('Erreur : ' . $e->getMessage());
}

3. Exécuter des requêtes (Query)

Pour des requêtes simples sans variables utilisateur :

PHP 
$reponse = $bdd->query('SELECT * FROM jeux_video');

while ($donnees = $reponse->fetch()) {
    echo $donnees['nom'] . "\n";
}

$reponse->closeCursor();

4. Requêtes Préparées

Pour la sécurité (éviter les injections SQL) et la performance :

PHP 
$possesseur = 'Michel';
$req = $bdd->prepare('SELECT nom, prix FROM jeux_video WHERE possesseur = ?');
$req->execute(array($possesseur));

while ($donnees = $req->fetch()) {
    echo $donnees['nom'] . ' coûte ' . $donnees['prix'] . ' euros' . "\n";
}

Marqueurs nommés

Plus lisible qu'avec les ?, les marqueurs nommés utilisent des :nom :

$req = $bdd->prepare('SELECT * FROM users WHERE email = :email AND age > :age');
$req->execute([
    ':email' => 'jean@mail.com',
    ':age' => 18
]);
$user = $req->fetch(PDO::FETCH_ASSOC);

CRUD complet

// CREATE (INSERT)
$req = $bdd->prepare('INSERT INTO users (nom, email) VALUES (:nom, :email)');
$req->execute([':nom' => 'Alice', ':email' => 'alice@mail.com']);
$lastId = $bdd->lastInsertId();

// READ (SELECT)
$req = $bdd->prepare('SELECT * FROM users WHERE id = ?');
$req->execute([$lastId]);
$user = $req->fetch(PDO::FETCH_ASSOC);

// UPDATE
$req = $bdd->prepare('UPDATE users SET nom = ? WHERE id = ?');
$req->execute(['Alice Dupont', $lastId]);

// DELETE
$req = $bdd->prepare('DELETE FROM users WHERE id = ?');
$req->execute([$lastId]);

Injection SQL — DANGER !

❌ JAMAIS de concaténation directe :

$bdd->query("SELECT * FROM users WHERE nom = '$_GET[nom]'")

✅ TOUJOURS des requêtes préparées :

$bdd->prepare("SELECT * FROM users WHERE nom = ?")

Mnémotechnique PDO

🎯 « query = lecture simple, prepare = avec variables »

  • query() → requêtes sans paramètres (SELECT * FROM ...)
  • prepare() + execute() → requêtes avec paramètres (WHERE id = ?)
  • fetchAll() → tous les résultats d'un coup
  • fetch() → un résultat à la fois (en boucle)
