1. Architecture MVC

Le pattern MVC (Modèle - Vue - Contrôleur) est l'architecture la plus répandue pour les applications web.

  • Modèle : Gère les données (BDD).
  • Vue : Gère l'affichage (HTML).
  • Contrôleur : Fait le lien entre Modèle et Vue (Logique).

2. Le Routeur

Le routeur est le point d'entrée de l'application (souvent index.php). Il analyse l'URL et appelle le bon contrôleur.

PHP 
// Simulation de l'URL : index.php?page=contact
$page = 'contact'; 

echo "Route demandée : " . $page . "\n";

if ($page === 'contact') {
    // $controller = new ContactController();
    // $controller->show();

    echo "Affichage de la vue 'contact.php'";
} else {
