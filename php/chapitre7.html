Introduction au Projet

Objectif du "Fil Rouge"

Nous allons créer une application complète de Gestion de Tâches (ToDo List). Ce projet va nous permettre d'appliquer concrètement l'architecture MVC, PDO, et la Programmation Orientée Objet.

Pré-requis : Pour ce chapitre, vous devez avoir un environnement serveur local (WAMP, XAMPP, MAMP ou Laragon) installé et lancé sur votre machine.

1. Préparation de la Base de Données

Ouvrez phpMyAdmin (généralement à l'adresse http://localhost/phpmyadmin) et exécutez le code SQL suivant pour créer la base et la table :

SQL 
-- Création de la base
CREATE DATABASE todo_app CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

USE todo_app;

-- Création de la table tasks
CREATE TABLE tasks (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    description TEXT,
    is_done TINYINT(1) DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- Données factices pour tester
INSERT INTO tasks (title, description) VALUES 
('Apprendre le PHP', 'Comprendre la syntaxe de base'),
('Maîtriser la POO', 'Classes, objets et héritage'),
('Faire le projet', 'Créer la ToDo List en MVC');

2. Structure des Dossiers (MVC)

Dans votre dossier www ou htdocs, créez un dossier todo-app. À l'intérieur, reproduisez cette structure :

  • todo-app/
    • app/ (Code métier)
      • Controllers/
      • Models/
      • Db.php
    • public/ (Accessible au navigateur)
      • css/
      • index.php
      • .htaccess
    • views/ (Fichiers HTML/PHP)

Exercice : Le Router (.htaccess)

Créez le fichier /public/.htaccess pour rediriger toutes les URLs vers index.php.

RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php [QSA,L]

3. Le Singleton de Connexion

Dans app/Db.php, nous allons créer une classe utilisant le pattern Singleton. Cela évite de créer 50 connexions à la base si on instancie 50 modèles.

PHP 
<?php
namespace App;

use PDO;
use PDOException;

class Db {
    private static $instance = null;
    private $pdo;

    private function __construct() {
        try {
            // Remplacez 'root' et '' par vos identifiants
            $this->pdo = new PDO('mysql:host=localhost;dbname=todo_app;charset=utf8', 'root', '');
            $this->pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
            $this->pdo->setAttribute(PDO::ATTR_DEFAULT_FETCH_MODE, PDO::FETCH_ASSOC);
        } catch (PDOException $e) {
            die("Erreur de connexion : " . $e->getMessage());
        }
    }

    public static function getInstance() {
        if (self::$instance === null) {
            self::$instance = new self();
        }
        return self::$instance->pdo;
    }
}

4. Le Modèle : Task

Le rôle du modèle est de parler à la BDD. Créez app/Models/Task.php.

PHP 
<?php
namespace App\Models;

use App\Db;

class Task {
    // Récupérer toutes les tâches
    public static function findAll() {
        $pdo = Db::getInstance();
        $stmt = $pdo->query("SELECT * FROM tasks ORDER BY created_at DESC");
        return $stmt->fetchAll();
    }
}

5. Contrôleur et Vue

Enfin, affichons le tout !

Le Contrôleur (app/Controllers/HomeController.php)

<?php
namespace App\Controllers;

use App\Models\Task;

class HomeController {
    public function index() {
        // 1. Demande les données au modèle
        $tasks = Task::findAll();
        
        // 2. Envoie les données à la vue
        require '../views/home.php';
    }
}

La Vue (views/home.php)

<!DOCTYPE html>
<html>
<head>
    <title>Ma ToDo List</title>
</head>
<body>
    <h1>Mes Tâches</h1>
    <ul>
        <?php foreach($tasks as $task): ?>
            <li>
                <strong><?= htmlspecialchars($task['title']) ?></strong>
                <?php if($task['is_done']): ?> (Fait) <?php endif; ?>
            </li>
        <?php endforeach; ?>
    </ul>
</body>
</html>

Le point d'entrée (public/index.php)

Pour finir, il nous faut notre fichier principal qui charge tout.

<?php
// Autoloader simple pour le cours (à remplacer par Composer un jour)
spl_autoload_register(function ($class) {
    // App\Controllers\HomeController -> app/Controllers/HomeController.php
    $prefix = 'App\\';
    $base_dir = __DIR__ . '/../app/';
    
    $len = strlen($prefix);
    if (strncmp($prefix, $class, $len) !== 0) return;
    
    $relative_class = substr($class, $len);
    $file = $base_dir . str_replace('\\', '/', $relative_class) . '.php';
    
    if (file_exists($file)) {
        require $file;
    }
});

use App\Controllers\HomeController;

// Route simple
$controller = new HomeController();
$controller->index();

Testez !

Ouvrez votre navigateur sur http://localhost/todo-app/public/. Vous devriez voir la liste des tâches que vous avez insérées dans la base de données !
