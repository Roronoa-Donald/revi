1. Ajouter une tâche (Create)

Pour ajouter des données, il nous faut un formulaire HTML et une méthode pour traiter l'envoi (POST).

1.1 Le Modèle (app/Models/Task.php)

// Ajouter dans la classe Task
public static function create($title) {
    $pdo = Db::getInstance();
    $stmt = $pdo->prepare("INSERT INTO tasks (title) VALUES (:title)");
    return $stmt->execute(['title' => $title]);
}

1.2 Le Contrôleur (app/Controllers/HomeController.php)

// Ajouter dans HomeController
public function add() {
    if (!empty($_POST['title'])) {
        $title = strip_tags($_POST['title']); // Sécurité basique
        Task::create($title);
    }
    // Redirection vers la liste
    header('Location: /todo-app/public/');
    exit;
}

1.3 La Vue (views/home.php)

Ajoutez ce formulaire en haut de votre body :

<form action="?action=add" method="post">
    <input type="text" name="title" placeholder="Nouvelle tâche..." required>
    <button type="submit">Ajouter</button>
</form>

2. Valider une tâche (Update)

Nous voulons pouvoir cocher une tâche comme "faite" ou "à faire".

A vous de jouer !

Essayez d'écrire la méthode SQL toggle($id) dans le Modèle pour inverser le statut is_done d'une tâche donnée.

// Dans Task.php
public static function toggle($id) {
    $pdo = Db::getInstance();
    $stmt = $pdo->prepare("UPDATE tasks SET is_done = NOT is_done WHERE id = :id");
    return $stmt->execute(['id' => $id]);
}

Ensuite, le contrôleur :

public function toggle() {
    if (isset($_GET['id'])) {
        Task::toggle($_GET['id']);
    }
    header('Location: /todo-app/public/');
    exit;
}

3. Supprimer (Delete)

Le principe est le même. On passe un ID, et on supprime la ligne.

// Model
public static function delete($id) {
    $pdo = Db::getInstance();
    $stmt = $pdo->prepare("DELETE FROM tasks WHERE id = :id");
    return $stmt->execute(['id' => $id]);
}

// Controller
public function delete() {
    if (isset($_GET['id'])) {
        Task::delete($_GET['id']);
    }
    header('Location: /todo-app/public/');
    exit;
}

4. Mise à jour du Routeur

Pour que nos formulaires (?action=add) fonctionnent, il faut update public/index.php.

// ... Autoloader ...
use App\Controllers\HomeController;
$controller = new HomeController();

// Routeur très basique
$action = $_GET['action'] ?? 'index';

if ($action === 'add') {
    $controller->add();
} elseif ($action === 'toggle') {
    $controller->toggle();
} elseif ($action === 'delete') {
    $controller->delete();
} else {
    $controller->index();
}
