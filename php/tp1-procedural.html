1

Variables & Types de données

En PHP, une variable commence toujours par $. PHP est dynamiquement typé : pas besoin de déclarer le type, il est déduit automatiquement.

Exemples de base
<?php
// Les types fondamentaux
$nom = "Alice";          // string (chaîne)
$age = 21;               // int (entier)
$moyenne = 14.5;         // float (décimal)
$estInscrit = true;      // bool (booléen)
$rien = null;            // null (vide)

// Affichage
echo "Nom : $nom, Age : $age";          // Guillemets doubles = interprète les variables
echo 'Nom : ' . $nom . ', Age : ' . $age; // Guillemets simples + concaténation avec .

// Vérifier un type
echo gettype($age);      // "integer"
echo is_string($nom);    // true (1)
?>

Micro-exercice 1.1 — Déclarer et afficher

Déclare 4 variables : $prenom (string), $note (float), $estPresent (bool) et $matiere (string). Affiche une phrase comme : "Donald a eu 15.5 en PHP".

Correction

<?php
// On déclare chaque variable avec le bon type de valeur
$prenom = "Donald";      // string : entre guillemets
$note = 15.5;            // float : nombre avec virgule (point en PHP!)
$estPresent = true;      // bool : true ou false, utile pour les conditions
$matiere = "PHP";        // string

// Affichage avec guillemets doubles (les $ sont interprétés)
echo "$prenom a eu $note en $matiere";
// Résultat : "Donald a eu 15.5 en PHP"
?>
💡 Pourquoi c'est important ?
À l'examen, on te donnera souvent un contexte ("on modélise un étudiant...") et tu devras choisir les bons types. float pour les notes/moyennes, string pour les noms, bool pour les états (inscrit, présent...).
2

Tableaux (Arrays)

Un tableau stocke plusieurs valeurs dans une seule variable. PHP a deux types : indexé (numéros) et associatif (clés nommées).

Tableaux indexés et associatifs
<?php
// Tableau indexé (indices 0, 1, 2...)
$fruits = ["Pomme", "Banane", "Orange"];
echo $fruits[0];  // "Pomme"
echo count($fruits); // 3

// Tableau associatif (clé => valeur)
$etudiant = [
    "nom" => "Dupont",
    "prenom" => "Alice",
    "note" => 14.5
];
echo $etudiant["nom"]; // "Dupont"

// Ajouter un élément
$fruits[] = "Kiwi";              // Ajout à la fin (indexé)
$etudiant["email"] = "a@b.com";  // Ajout avec clé (associatif)

// Parcourir un tableau associatif
foreach ($etudiant as $cle => $valeur) {
    echo "$cle : $valeur\n";
}
?>

Micro-exercice 1.2 — Tableau de notes

Crée un tableau associatif $notes avec 3 matières et leurs notes (ex: "PHP" => 15, "C#" => 12, "SQL" => 17). Parcours-le avec foreach pour afficher chaque matière et sa note.

Correction

<?php
// Tableau associatif : clé (string) => valeur (int)
$notes = [
    "PHP" => 15,   // La clé est le nom de la matière
    "C#"  => 12,   // La valeur est la note
    "SQL" => 17
];

// foreach avec $cle => $valeur pour accéder aux deux
foreach ($notes as $matiere => $note) {
    echo "Matière : $matiere — Note : $note/20\n";
}
// Résultat :
// Matière : PHP — Note : 15/20
// Matière : C# — Note : 12/20
// Matière : SQL — Note : 17/20
?>
💡 Pourquoi c'est important ?
Les tableaux associatifs servent à modéliser des données avant de passer à la POO. Le prof te demandera souvent de stocker des informations (notes, étudiants...) et de les parcourir. foreach ($tab as $cle => $val) est LA syntaxe à connaître par cœur.
3

Structures de contrôle

if / else / elseif
<?php
$note = 14;

if ($note >= 16) {
    echo "Très bien";
} elseif ($note >= 14) {
    echo "Bien";          // ← Celui-ci s'exécute
} elseif ($note >= 12) {
    echo "Assez bien";
} else {
    echo "À améliorer";
}
?>
Boucles : for, while, foreach
<?php
// for : quand on connaît le nombre d'itérations
for ($i = 1; $i <= 5; $i++) {
    echo "Tour $i\n";
}

// while : quand la condition peut changer
$compteur = 0;
while ($compteur < 3) {
    echo "Compteur : $compteur\n";
    $compteur++;
}

// foreach : TOUJOURS pour parcourir un tableau
$prenoms = ["Alice", "Bob", "Charlie"];
foreach ($prenoms as $prenom) {
    echo "Bonjour $prenom !\n";
}
?>

Micro-exercice 1.3 — Mentions automatiques

Écris un code qui prend un tableau de notes $notes = [8, 12, 15, 17, 9, 14] et affiche pour chaque note la mention correspondante :
≥16 → "Très bien" | ≥14 → "Bien" | ≥12 → "Assez bien" | ≥10 → "Passable" | <10 → "Insuffisant"

Correction

<?php
$notes = [8, 12, 15, 17, 9, 14];

// foreach car on parcourt un tableau
foreach ($notes as $note) {
    // if/elseif : du plus grand au plus petit
    if ($note >= 16) {
        $mention = "Très bien";
    } elseif ($note >= 14) {
        $mention = "Bien";
    } elseif ($note >= 12) {
        $mention = "Assez bien";
    } elseif ($note >= 10) {
        $mention = "Passable";
    } else {
        $mention = "Insuffisant";
    }
    echo "Note $note/20 → $mention\n";
}
?>
💡 Pourquoi c'est important ?
Ce pattern (foreach + if/elseif) revient constamment à l'examen. Le prof adore les systèmes de mention/catégorisation. Attention : l'ordre des elseif est crucial — toujours du plus restrictif au moins restrictif !
4

Fonctions

Une fonction est un bloc de code réutilisable. Elle peut prendre des paramètres et retourner un résultat.

Fonctions : déclaration, paramètres, retour
<?php
// Fonction simple sans retour
function saluer($nom) {
    echo "Bonjour $nom !\n";
}
saluer("Alice");  // "Bonjour Alice !"

// Fonction avec retour
function calculerMoyenne($notes) {
    $somme = 0;
    foreach ($notes as $note) {
        $somme += $note;       // $somme = $somme + $note
    }
    return $somme / count($notes);  // Retourne le résultat
}

$mesNotes = [12, 15, 8, 17];
$moy = calculerMoyenne($mesNotes);
echo "Moyenne : $moy";  // Moyenne : 13

// Paramètre avec valeur par défaut
function afficherNote($note, $sur = 20) {
    echo "$note/$sur\n";
}
afficherNote(15);      // "15/20" (défaut)
afficherNote(15, 100); // "15/100"
?>

Micro-exercice 1.4 — Fonction getMention()

Crée une fonction getMention($note) qui retourne (pas echo !) la mention selon les mêmes critères que l'exercice précédent. Teste-la avec echo getMention(15);

Correction

<?php
// La fonction RETOURNE une valeur au lieu de l'afficher
// Avantage : on peut réutiliser le résultat ailleurs
function getMention($note) {
    if ($note >= 16) {
        return "Très bien";   // return = renvoie ET sort de la fonction
    } elseif ($note >= 14) {
        return "Bien";
    } elseif ($note >= 12) {
        return "Assez bien";
    } elseif ($note >= 10) {
        return "Passable";
    } else {
        return "Insuffisant";
    }
}

// Test
echo getMention(15);   // "Bien"
echo getMention(8);    // "Insuffisant"

// Avantage de return : on peut stocker et manipuler
$mention = getMention(17);
echo "Mention obtenue : $mention";  // "Mention obtenue : Très bien"
?>
💡 Pourquoi return et pas echo ?
echo affiche directement → on ne peut pas réutiliser.
return renvoie la valeur → on peut la stocker, la comparer, la passer à une autre fonction.
Règle d'examen : si on te dit "la méthode retourne...", utilise return. Si on te dit "la méthode affiche...", utilise echo.

Exercice de Synthèse — Bulletin de notes 5 pts

Contexte : On souhaite gérer le bulletin de notes d'un étudiant.

  1. (1 pt) Déclare un tableau associatif $etudiant contenant : nom, prénom, et un sous-tableau notes avec au moins 4 matières et leurs notes.
  2. (2 pts) Crée une fonction calculerMoyenne($notes) qui prend un tableau de notes et retourne la moyenne.
  3. (1 pt) Crée une fonction getMention($moyenne) qui retourne la mention.
  4. (1 pt) Affiche le bulletin complet : nom, chaque matière/note, moyenne générale et mention.

Correction détaillée

<?php
// ============================================================
// QUESTION 1 : Tableau associatif avec sous-tableau
// ============================================================
$etudiant = [
    "nom"     => "Dupont",
    "prenom"  => "Alice",
    "notes"   => [          // Sous-tableau associatif
        "PHP"  => 15,
        "C#"   => 12,
        "SQL"  => 17,
        "UML"  => 14
    ]
];

// ============================================================
// QUESTION 2 : Fonction calculerMoyenne
// ============================================================
function calculerMoyenne($notes) {
    $somme = 0;
    $count = 0;
    // On parcourt les notes (valeurs du tableau associatif)
    foreach ($notes as $note) {
        $somme += $note;
        $count++;
    }
    // Alternative : return array_sum($notes) / count($notes);
    return $somme / $count;
}

// ============================================================
// QUESTION 3 : Fonction getMention
// ============================================================
function getMention($moyenne) {
    if ($moyenne >= 16) return "Très bien";
    elseif ($moyenne >= 14) return "Bien";
    elseif ($moyenne >= 12) return "Assez bien";
    elseif ($moyenne >= 10) return "Passable";
    else return "Insuffisant";
}

// ============================================================
// QUESTION 4 : Affichage du bulletin
// ============================================================
echo "=== BULLETIN DE NOTES ===\n";
echo "Étudiant : " . $etudiant["prenom"] . " " . $etudiant["nom"] . "\n";
echo "-------------------------\n";

foreach ($etudiant["notes"] as $matiere => $note) {
    echo "$matiere : $note/20\n";
}

$moyenne = calculerMoyenne($etudiant["notes"]);
$mention = getMention($moyenne);

echo "-------------------------\n";
echo "Moyenne générale : " . round($moyenne, 2) . "/20\n";
echo "Mention : $mention\n";
?>
💡 Ce que cet exercice teste (et ce que le prof note) :
✅ Savoir structurer des données avec des tableaux associatifs (Q1)
✅ Écrire une fonction avec paramètre et return (Q2)
✅ Utiliser if/elseif correctement (Q3)
✅ Combiner foreach + fonctions pour un résultat final (Q4)
