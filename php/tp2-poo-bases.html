TP 2 : Créer des Classes et des Objets

C'est LE sujet central de l'examen. On passe du procédural à l'objet : au lieu de tableaux, on utilise des classes pour modéliser le monde réel.

🏗️ Analogie : Une classe = un plan d'architecte. Un objet = la maison construite à partir du plan. On peut construire plusieurs maisons (objets) à partir du même plan (classe).
1

Déclarer une classe

Structure d'une classe PHP
<?php
class Livre {
    // ATTRIBUTS (= les données que chaque livre possède)
    public $titre;
    public $auteur;
    public $nbPages;
}

// Créer un OBJET (= une instance de la classe)
$monLivre = new Livre();

// Accéder aux attributs avec ->
$monLivre->titre = "PHP pour les nuls";
$monLivre->auteur = "Jean Dupont";
$monLivre->nbPages = 350;

echo $monLivre->titre;  // "PHP pour les nuls"
?>

Règle de nommage : Le nom d'une classe commence toujours par une majuscule (PascalCase) : Livre, Etudiant, CompteBancaire.

Micro-exercice 2.1 — Première classe

Crée une classe Etudiant avec 3 attributs publics : $nom, $prenom, $age. Instancie un objet et affiche son nom.

Correction

<?php
// class = on définit le "moule"
class Etudiant {
    public $nom;      // public = accessible de partout
    public $prenom;   // On verra dans le TP3 pourquoi c'est risqué
    public $age;
}

// new = on fabrique un objet à partir du moule
$etu = new Etudiant();

// -> = on accède à un attribut/méthode de l'objet
$etu->nom = "Dupont";
$etu->prenom = "Alice";
$etu->age = 21;

echo $etu->prenom . " " . $etu->nom; // "Alice Dupont"
?>
💡 Vocabulaire d'examen :
• class Etudiant → on dit "déclarer/définir la classe"
• new Etudiant() → on dit "instancier un objet" ou "créer une instance"
• $etu → c'est "un objet" ou "une instance de Etudiant"
• $nom → c'est "un attribut" ou "une propriété"
2

Le Constructeur : __construct()

Le constructeur est une méthode spéciale appelée automatiquement quand on fait new. Il sert à initialiser les attributs dès la création.

Sans constructeur vs Avec constructeur
<?php
// ❌ SANS constructeur — il faut définir chaque attribut manuellement
$livre1 = new Livre();
$livre1->titre = "PHP";
$livre1->auteur = "Jean";
$livre1->nbPages = 300;

// ✅ AVEC constructeur — tout est fait en une ligne
class Livre {
    public $titre;
    public $auteur;
    public $nbPages;

    // __construct est appelé automatiquement par new
    public function __construct($titre, $auteur, $nbPages) {
        $this->titre = $titre;       // $this = "moi-même", l'objet en cours
        $this->auteur = $auteur;
        $this->nbPages = $nbPages;
    }
}

// Maintenant, on crée + initialise en une seule ligne
$livre1 = new Livre("PHP POO", "Jean Dupont", 350);
$livre2 = new Livre("C# Avancé", "Marie Martin", 420);

echo $livre1->titre;  // "PHP POO"
echo $livre2->titre;  // "C# Avancé"
?>
⚠️ Piège d'examen : $this->titre (l'attribut de l'objet) ≠ $titre (le paramètre de la fonction). Si tu écris $titre = $titre; au lieu de $this->titre = $titre;, ça ne marche pas !

Micro-exercice 2.2 — Constructeur Etudiant

Reprends la classe Etudiant et ajoute un constructeur qui prend $nom, $prenom, $age. Crée 2 étudiants en une seule ligne chacun.

Correction

<?php
class Etudiant {
    public $nom;
    public $prenom;
    public $age;

    // Constructeur : reçoit les valeurs et les stocke dans l'objet
    public function __construct($nom, $prenom, $age) {
        $this->nom = $nom;       // $this->nom = attribut de l'objet
        $this->prenom = $prenom; // $prenom = paramètre reçu
        $this->age = $age;
    }
}

// Création : les arguments passent directement au __construct
$etu1 = new Etudiant("Dupont", "Alice", 21);
$etu2 = new Etudiant("Martin", "Bob", 22);

echo $etu1->prenom; // "Alice"
echo $etu2->prenom; // "Bob"
?>
3

Méthodes (= fonctions dans une classe)

Une méthode est une fonction définie dans une classe. Elle utilise $this pour accéder aux attributs de l'objet.

Méthodes : afficher, calculer, retourner
<?php
class Livre {
    public $titre;
    public $auteur;
    public $nbPages;

    public function __construct($titre, $auteur, $nbPages) {
        $this->titre = $titre;
        $this->auteur = $auteur;
        $this->nbPages = $nbPages;
    }

    // Méthode qui AFFICHE (echo)
    public function afficherInfos() {
        echo "Titre : {$this->titre}, ";
        echo "Auteur : {$this->auteur}, ";
        echo "Pages : {$this->nbPages}\n";
    }

    // Méthode qui RETOURNE une valeur
    public function estLong() {
        return $this->nbPages > 300;  // Retourne true ou false
    }
}

$livre = new Livre("PHP POO", "Jean", 450);
$livre->afficherInfos();  // Titre : PHP POO, Auteur : Jean, Pages : 450

if ($livre->estLong()) {
    echo "C'est un gros livre !";
}
?>

Micro-exercice 2.3 — La méthode afficherInfos()

Dans la classe Etudiant, ajoute une méthode afficherInfos() qui affiche : "Nom : Dupont, Prénom : Alice, Age : 21 ans". Ajoute aussi un attribut $notes (tableau) initialisé à vide dans le constructeur, et une méthode ajouterNote($note).

Correction

<?php
class Etudiant {
    public $nom;
    public $prenom;
    public $age;
    public $notes;  // Tableau de notes

    public function __construct($nom, $prenom, $age) {
        $this->nom = $nom;
        $this->prenom = $prenom;
        $this->age = $age;
        $this->notes = [];  // Initialisé à tableau vide
    }

    // Méthode d'affichage
    public function afficherInfos() {
        echo "Nom : {$this->nom}, ";
        echo "Prénom : {$this->prenom}, ";
        echo "Age : {$this->age} ans\n";
    }

    // Méthode pour ajouter une note au tableau
    public function ajouterNote($note) {
        $this->notes[] = $note;  // [] = ajouter à la fin du tableau
    }
}

$etu = new Etudiant("Dupont", "Alice", 21);
$etu->afficherInfos();
$etu->ajouterNote(15);
$etu->ajouterNote(12);
echo count($etu->notes); // 2
?>
💡 Pourquoi $this ?
$this représente "l'objet en cours". Quand tu appelles $etu->ajouterNote(15), à l'intérieur de la méthode, $this = $etu. Si tu appelles $etu2->ajouterNote(18), $this = $etu2. Chaque objet a ses propres données !
4

Méthodes de calcul et __toString()

calculerMoyenne() et __toString()
<?php
class Etudiant {
    public $nom;
    public $prenom;
    public $notes;

    public function __construct($nom, $prenom) {
        $this->nom = $nom;
        $this->prenom = $prenom;
        $this->notes = [];
    }

    public function ajouterNote($note) {
        $this->notes[] = $note;
    }

    // Méthode de calcul qui RETOURNE la moyenne
    public function calculerMoyenne() {
        if (count($this->notes) === 0) {
            return 0;  // Éviter la division par zéro !
        }
        return array_sum($this->notes) / count($this->notes);
    }

    // Méthode magique : appelée quand on fait echo $objet
    public function __toString() {
        $moy = round($this->calculerMoyenne(), 2);
        return "{$this->prenom} {$this->nom} — Moyenne : $moy/20";
    }
}

$etu = new Etudiant("Dupont", "Alice");
$etu->ajouterNote(15);
$etu->ajouterNote(12);
$etu->ajouterNote(17);

echo $etu->calculerMoyenne(); // 14.666...
echo $etu;  // "Alice Dupont — Moyenne : 14.67/20" (appelle __toString)
?>

Micro-exercice 2.4 — getMention() dans la classe

Ajoute à la classe Etudiant une méthode getMention() qui retourne la mention basée sur la moyenne (même critères que TP1). Teste en faisant echo $etu->getMention();

Correction

<?php
// Dans la classe Etudiant, on ajoute :
public function getMention() {
    $moy = $this->calculerMoyenne(); // On réutilise notre propre méthode !

    if ($moy >= 16) return "Très bien";
    elseif ($moy >= 14) return "Bien";
    elseif ($moy >= 12) return "Assez bien";
    elseif ($moy >= 10) return "Passable";
    else return "Insuffisant";
}

// Test
$etu = new Etudiant("Dupont", "Alice");
$etu->ajouterNote(15);
$etu->ajouterNote(12);
$etu->ajouterNote(17);
echo $etu->getMention(); // "Bien" (moyenne = 14.67)
?>
💡 Avantage vs procédural :
En TP1, getMention() prenait la note en paramètre. Maintenant, la méthode utilise $this->calculerMoyenne() — elle accède directement aux données de l'objet ! C'est ça la POO : les données et les actions sont regroupées dans le même objet.

Exercice de Synthèse — Bibliothèque 5 pts

Contexte : On souhaite modéliser une bibliothèque avec des livres. (Inspiré de l'épreuve 2024/2025)

  1. (2 pts) Créez une classe Livre avec les attributs : titre (chaîne), auteur (chaîne), nbPages (entier). Implémentez un constructeur permettant d'initialiser ces attributs.
  2. (1 pt) Ajoutez une méthode afficherInfos() qui affiche : "Titre : <titre>, Auteur : <auteur>, Nombre de pages : <nbPages>"
  3. (2 pts) Instanciez deux objets de la classe Livre et affichez leurs informations.

Correction détaillée — C'est exactement ce style à l'examen !

<?php
// ===========================================================
// QUESTION 1 (2 pts) : Classe Livre + Constructeur
// ===========================================================
class Livre {
    // Attributs : les caractéristiques d'un livre
    public $titre;      // string
    public $auteur;     // string
    public $nbPages;    // int

    // Constructeur : initialise les 3 attributs
    // Appelé automatiquement par new Livre(...)
    public function __construct($titre, $auteur, $nbPages) {
        $this->titre = $titre;
        $this->auteur = $auteur;
        $this->nbPages = $nbPages;
    }
    // Barème estimé :
    // - Déclaration class + attributs : 1 pt
    // - __construct correct avec $this : 1 pt

    // ===========================================================
    // QUESTION 2 (1 pt) : Méthode afficherInfos()
    // ===========================================================
    public function afficherInfos() {
        echo "Titre : {$this->titre}, ";
        echo "Auteur : {$this->auteur}, ";
        echo "Nombre de pages : {$this->nbPages}\n";
    }
    // Barème : echo correct avec $this-> : 1 pt
}

// ===========================================================
// QUESTION 3 (2 pts) : Instancier et afficher
// ===========================================================

// Créer 2 objets avec new
$livre1 = new Livre("Les Misérables", "Victor Hugo", 1500);
$livre2 = new Livre("Le Petit Prince", "Saint-Exupéry", 96);

// Appeler afficherInfos() sur chaque objet
$livre1->afficherInfos();
// Affiche : Titre : Les Misérables, Auteur : Victor Hugo, Nombre de pages : 1500

$livre2->afficherInfos();
// Affiche : Titre : Le Petit Prince, Auteur : Saint-Exupéry, Nombre de pages : 96

// Barème :
// - 2 instanciations correctes : 1 pt
// - Appels afficherInfos() : 1 pt
?>
💡 Ce que le prof note :
✅ Le mot-clé class et le nom en PascalCase
✅ Les attributs déclarés avec public (ou private au TP3)
✅ __construct() avec les paramètres corrects
✅ $this->attribut = $parametre (pas l'inverse !)
✅ new Classe(...) pour instancier
✅ $objet->methode() pour appeler une méthode
Le TP3 reprendra exactement cette classe pour l'encapsuler.
