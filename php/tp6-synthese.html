Exercice 1 — Création et manipulation de classes 6 pts

Contexte : On souhaite modéliser un système de gestion d'étudiants et de leurs notes.

  1. (2 pts) Créez une classe Etudiant avec les attributs suivants : nom (chaîne), prenom (chaîne), notes (tableau). Implémentez un constructeur qui initialise les nom/prénom et crée un tableau de notes vide.
  2. (1 pt) Ajoutez une méthode ajouterNote($note) qui ajoute une note au tableau.
  3. (2 pts) Ajoutez une méthode calculerMoyenne() qui retourne la moyenne des notes (retourne 0 si aucune note).
  4. (1 pt) Ajoutez une méthode afficherBulletin() qui affiche le nom, prénom, toutes les notes et la moyenne.

Corrigé Exercice 1

<?php
class Etudiant {
    // Attributs (public pour l'instant, seront private à l'exo 2)
    public $nom;        // string
    public $prenom;     // string
    public $notes;      // array

    // Q1 (2pts) : Constructeur
    public function __construct($nom, $prenom) {
        $this->nom = $nom;
        $this->prenom = $prenom;
        $this->notes = [];    // Tableau vide à la création
    }

    // Q2 (1pt) : Ajouter une note
    public function ajouterNote($note) {
        $this->notes[] = $note;  // Ajoute à la fin du tableau
    }

    // Q3 (2pts) : Calculer la moyenne
    public function calculerMoyenne() {
        // Protection contre la division par zéro
        if (count($this->notes) === 0) {
            return 0;
        }
        return array_sum($this->notes) / count($this->notes);
    }

    // Q4 (1pt) : Afficher le bulletin
    public function afficherBulletin() {
        echo "=== BULLETIN ===\n";
        echo "Étudiant : {$this->prenom} {$this->nom}\n";
        echo "Notes : ";
        foreach ($this->notes as $note) {
            echo "$note ";
        }
        echo "\nMoyenne : " . round($this->calculerMoyenne(), 2) . "/20\n";
    }
}

// Test
$etu = new Etudiant("Dupont", "Alice");
$etu->ajouterNote(15);
$etu->ajouterNote(12);
$etu->ajouterNote(17);
$etu->ajouterNote(9);
$etu->afficherBulletin();
// === BULLETIN ===
// Étudiant : Alice Dupont
// Notes : 15 12 17 9
// Moyenne : 13.25/20
?>
💡 Points clés à ne pas oublier :
• $this->notes = [] dans le constructeur (sinon pas initialisé)
• count() === 0 avant de diviser (le prof enlève des points si tu oublies)
• round() pour un affichage propre de la moyenne

Exercice 2 — Encapsulation et validation 6 pts

Contexte : On veut sécuriser la gestion des données de l'étudiant.

  1. (1 pt) Modifiez la classe Etudiant pour que tous les attributs soient privés.
  2. (2 pts) Ajoutez les getters nécessaires : getNom(), getPrenom(), getNotes(), getMoyenne().
  3. (2 pts) Modifiez ajouterNote($note) pour qu'elle refuse les notes qui ne sont pas entre 0 et 20 (affichez un message d'erreur).
  4. (1 pt) Ajoutez une méthode getMention() qui retourne la mention selon la moyenne :
    ≥16 : "Très bien" | ≥14 : "Bien" | ≥12 : "Assez bien" | ≥10 : "Passable" | <10 : "Insuffisant"

Corrigé Exercice 2

<?php
class Etudiant {
    // Q1 (1pt) : Tous les attributs en private
    private $nom;
    private $prenom;
    private $notes;

    public function __construct($nom, $prenom) {
        $this->nom = $nom;
        $this->prenom = $prenom;
        $this->notes = [];
    }

    // Q2 (2pts) : Getters
    public function getNom()     { return $this->nom; }
    public function getPrenom()  { return $this->prenom; }
    public function getNotes()   { return $this->notes; }
    public function getMoyenne() { return $this->calculerMoyenne(); }

    // Q3 (2pts) : ajouterNote avec validation
    public function ajouterNote($note) {
        // Vérification : doit être numérique ET entre 0 et 20
        if (!is_numeric($note) || $note < 0 || $note > 20) {
            echo "Erreur : la note doit être entre 0 et 20 (reçu: $note)\n";
            return;    // On sort sans ajouter
        }
        $this->notes[] = $note;
    }

    private function calculerMoyenne() {
        if (count($this->notes) === 0) return 0;
        return array_sum($this->notes) / count($this->notes);
    }

    // Q4 (1pt) : Mention
    public function getMention() {
        $moy = $this->calculerMoyenne();
        if ($moy >= 16) return "Très bien";
        elseif ($moy >= 14) return "Bien";
        elseif ($moy >= 12) return "Assez bien";
        elseif ($moy >= 10) return "Passable";
        else return "Insuffisant";
    }

    public function afficherBulletin() {
        echo "=== BULLETIN ===\n";
        echo "Étudiant : {$this->prenom} {$this->nom}\n";
        foreach ($this->notes as $note) {
            echo "  - $note/20\n";
        }
        echo "Moyenne : " . round($this->calculerMoyenne(), 2) . "/20\n";
        echo "Mention : " . $this->getMention() . "\n";
    }
}

// Tests
$etu = new Etudiant("Martin", "Bob");
$etu->ajouterNote(16);     // ✅
$etu->ajouterNote(14);     // ✅
$etu->ajouterNote(-5);     // ❌ Erreur
$etu->ajouterNote(25);     // ❌ Erreur
$etu->ajouterNote(18);     // ✅
$etu->afficherBulletin();
// Mention : Bien (moyenne 16)

echo $etu->getNom();       // ✅ "Martin"
// echo $etu->nom;         // ❌ ERREUR (private)
?>

Exercice 3 — Héritage, Polymorphisme & Exceptions 7 pts

Contexte : On veut étendre le système avec des étudiants boursiers et une gestion d'erreurs robuste.

  1. (1 pt) Créez une exception personnalisée NoteInvalideException.
  2. (1 pt) Modifiez la méthode ajouterNote() pour qu'elle lance NoteInvalideException au lieu d'afficher un message.
  3. (2 pts) Créez une classe EtudiantBoursier qui hérite de Etudiant et ajoute un attribut privé $montantBourse. Implémentez le constructeur avec parent::__construct().
  4. (2 pts) Redéfinissez afficherBulletin() dans EtudiantBoursier pour inclure le montant de la bourse.
  5. (1 pt) Écrivez un programme de test avec try/catch qui ajoute des notes (valides et invalides) à un EtudiantBoursier et affiche son bulletin.

Corrigé Exercice 3

<?php
// ===========================================================
// Q1 (1pt) : Exception personnalisée
// ===========================================================
class NoteInvalideException extends Exception {}

// ===========================================================
// Q2 (1pt) : Modification de ajouterNote
// ===========================================================
class Etudiant {
    private $nom;
    private $prenom;
    private $notes;

    public function __construct($nom, $prenom) {
        $this->nom = $nom;
        $this->prenom = $prenom;
        $this->notes = [];
    }

    public function getNom()     { return $this->nom; }
    public function getPrenom()  { return $this->prenom; }
    public function getNotes()   { return $this->notes; }

    // Modifié : lance une exception au lieu d'echo
    public function ajouterNote($note) {
        if (!is_numeric($note) || $note < 0 || $note > 20) {
            throw new NoteInvalideException(
                "Note invalide : $note (doit être entre 0 et 20)"
            );
        }
        $this->notes[] = $note;
    }

    public function calculerMoyenne() {
        if (count($this->notes) === 0) return 0;
        return array_sum($this->notes) / count($this->notes);
    }

    public function getMention() {
        $moy = $this->calculerMoyenne();
        if ($moy >= 16) return "Très bien";
        elseif ($moy >= 14) return "Bien";
        elseif ($moy >= 12) return "Assez bien";
        elseif ($moy >= 10) return "Passable";
        else return "Insuffisant";
    }

    public function afficherBulletin() {
        echo "=== BULLETIN ===\n";
        echo "Étudiant : {$this->prenom} {$this->nom}\n";
        foreach ($this->notes as $note) {
            echo "  - $note/20\n";
        }
        $moy = round($this->calculerMoyenne(), 2);
        echo "Moyenne : $moy/20 — Mention : " . $this->getMention() . "\n";
    }
}

// ===========================================================
// Q3 (2pts) : EtudiantBoursier hérite de Etudiant
// ===========================================================
class EtudiantBoursier extends Etudiant {
    private $montantBourse;

    public function __construct($nom, $prenom, $montantBourse) {
        parent::__construct($nom, $prenom);  // Initialise nom, prenom, notes
        $this->montantBourse = $montantBourse;
    }

    public function getMontantBourse() {
        return $this->montantBourse;
    }

    // ===========================================================
    // Q4 (2pts) : Redéfinition de afficherBulletin
    // ===========================================================
    public function afficherBulletin() {
        parent::afficherBulletin();   // Réutilise l'affichage du parent
        echo "Bourse : {$this->montantBourse}€/an\n";
    }
}

// ===========================================================
// Q5 (1pt) : Programme de test avec try/catch
// ===========================================================
$boursier = new EtudiantBoursier("Diallo", "Fatou", 5000);

$notesAajouter = [14, 16, 25, 11, -3, 18, 15];

foreach ($notesAajouter as $note) {
    try {
        $boursier->ajouterNote($note);
        echo "Note $note ajoutée ✅\n";
    } catch (NoteInvalideException $e) {
        echo "Rejetée ❌ : " . $e->getMessage() . "\n";
    }
}

echo "\n";
$boursier->afficherBulletin();
// === BULLETIN ===
// Étudiant : Fatou Diallo
//   - 14/20
//   - 16/20
//   - 11/20
//   - 18/20
//   - 15/20
// Moyenne : 14.8/20 — Mention : Bien
// Bourse : 5000€/an
?>

    return $this->attr;

    if (/* validation */) {
        $this->attr = $v;
    }

    public function __construct($a, $b) {
        parent::__construct($a);
        $this->propre = $b;
    }
}
