Cheatsheet HTML & JavaScript

Revision rapide, accessible sans mot de passe, pour revoir les balises, attributs, classes et notions JavaScript utilisees dans les 10 projets.

Structure HTML

  • <main>: contenu principal de la page.
  • <section>: partie thematique.
  • <article>: bloc autonome, comme une carte.
  • <header>: entete d'une page ou d'un bloc.
  • <div>: conteneur neutre pour organiser.

Textes et medias

  • <h1> a <h2>: titres hierarchiques.
  • <p>: paragraphe.
  • <img src="" alt="">: image avec texte alternatif.
  • <a href="">: lien.
  • <button>: action cliquable.

Listes

  • <ul>: liste non ordonnee.
  • <ol>: liste ordonnee.
  • <li>: element de liste.
<ul class="skill-list">

Formulaires

  • <form>: regroupe les champs.
  • <label for="">: nom visible du champ.
  • <input id="" type="">: champ court.
  • <textarea>: texte long.
  • <select> et <option>: liste de choix.

Types input

  • text: texte simple.
  • email: adresse email.
  • password: mot de passe masque.
  • number: nombre.
  • checkbox: choix multiple.
  • radio: choix unique avec meme name.
  • date: date.

Classes et attributs

  • class: relie ton HTML au CSS fourni.
  • id: identifiant unique, utile en JavaScript.
  • data-*: stocke une info lisible avec dataset.
  • placeholder: exemple dans un champ.
  • required: champ obligatoire.

JavaScript essentiel

let score = 0;

function increment() {
  score = score + 1;
  document.getElementById('countValue').textContent = score;
}

document.getElementById('incrementBtn').addEventListener('click', increment);

let cree une variable, function regroupe une action, getElementById selectionne un element, textContent change le texte visible, addEventListener reagit a un evenement.

DOM utile

  • document.querySelector(): selectionne le premier element.
  • document.querySelectorAll(): selectionne plusieurs elements.
  • document.createElement(): cree une balise.
  • appendChild(): ajoute une balise dans une autre.

Mini reflexe debug

  • Si le style ne marche pas: verifie la classe exacte.
  • Si JS ne marche pas: verifie l'id appele dans le code.
  • Si un clic ne fait rien: verifie addEventListener.
  • Si un calcul donne un resultat bizarre: utilise Number().
