Chapitre 1 Probabilités et Bayes

Probabilités de Base

Probabilité d'un événement

$$P(A) = \frac{\text{Nombre de cas favorables}}{\text{Nombre de cas possibles}}$$

Valable uniquement en équiprobabilité

Événement contraire

$$P(\bar{A}) = 1 - P(A)$$

$\bar{A}$ est l'événement "non A"

Union de deux événements

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

$A \cup B$ = "A ou B" (au moins l'un des deux)

Événements incompatibles

$$\text{Si } A \cap B = \emptyset \text{, alors } P(A \cup B) = P(A) + P(B)$$

Deux événements qui ne peuvent pas se produire en même temps

Probabilités Conditionnelles et Bayes

Probabilité conditionnelle

$$P_A(B) = P(B|A) = \frac{P(A \cap B)}{P(A)}$$

Probabilité de B sachant que A s'est réalisé

Intersection (règle de la chaîne)

$$P(A \cap B) = P(A) \times P_A(B)$$

Probabilité que A ET B se réalisent

Formule des probabilités totales

$$P(B) = \sum_{i=1}^{n} P(A_i) \times P_{A_i}(B)$$

$(A_i)$ forme une partition de l'univers

Formule de Bayes

$$P_B(A) = \frac{P(A) \times P_A(B)}{P(B)}$$

Permet d'inverser le conditionnement

Indépendance

$$A \text{ et } B \text{ indépendants} \Leftrightarrow P(A \cap B) = P(A) \times P(B)$$

La réalisation de l'un n'affecte pas l'autre

Chapitre 2 Dénombrement

Factorielle

$$n! = n \times (n-1) \times \cdots \times 2 \times 1$$

$0! = 1$ par convention

Arrangements (ordre important)

$$A_n^k = \frac{n!}{(n-k)!}$$

Choix ordonné de k éléments parmi n

Permutations

$$P_n = n!$$

Nombre de façons d'ordonner n éléments

Combinaisons (ordre non important)

$$C_n^k = \frac{n!}{k!(n-k)!}$$

Choix non ordonné de k éléments parmi n

Symétrie des combinaisons

$$C_n^k = C_n^{n-k}$$

Choisir k éléments = exclure n-k éléments

Triangle de Pascal

$$C_n^k = C_{n-1}^{k-1} + C_{n-1}^{k}$$

Relation de récurrence

Chapitre 3 Variables Aléatoires Discrètes

Espérance

$$E(X) = \sum_{i} x_i \cdot P(X = x_i)$$

Moyenne pondérée des valeurs par leurs probabilités

Variance

$$V(X) = E(X^2) - [E(X)]^2$$

Mesure la dispersion autour de l'espérance

Écart-type

$$\sigma(X) = \sqrt{V(X)}$$

Dispersion dans la même unité que X

Linéarité de l'espérance

$$E(aX + b) = aE(X) + b$$

L'espérance est linéaire

Variance et transformation

$$V(aX + b) = a^2 V(X)$$

La constante additive n'affecte pas la variance

Loi Binomiale $X \sim \mathcal{B}(n, p)$

Probabilité binomiale

$$P(X = k) = C_n^k \cdot p^k \cdot (1-p)^{n-k}$$

k succès parmi n épreuves indépendantes

Espérance binomiale

$$E(X) = np$$

Nombre moyen de succès attendus

Variance binomiale

$$V(X) = np(1-p)$$

Maximale quand p = 0.5

Chapitre 4 Variables Aléatoires Continues — Dérivées & Primitives

Ces formules sont indispensables pour le chapitre sur les variables aléatoires continues. Voir le cours complet avec exercices →

Puissances

$$(x^n)' = nx^{n-1}$$ $$\int x^n dx = \frac{x^{n+1}}{n+1} \quad (n \neq -1)$$

Inverse

$$\left(\frac{1}{x}\right)' = -\frac{1}{x^2}$$ $$\int \frac{1}{x} dx = \ln|x|$$

Exponentielle

$$(e^x)' = e^x$$ $$\int e^x dx = e^x$$

Exponentielle composée

$$(e^{ax})' = ae^{ax}$$ $$\int e^{ax} dx = \frac{1}{a}e^{ax}$$

Logarithme

$$(\ln x)' = \frac{1}{x}$$ $$\int \ln x \, dx = x\ln x - x$$

Racine carrée

$$(\sqrt{x})' = \frac{1}{2\sqrt{x}}$$ $$\int \sqrt{x} \, dx = \frac{2}{3}x\sqrt{x}$$

Intégration par Parties

Formule IPP

$$\int u \cdot v' = uv - \int u' \cdot v$$

Règle LIATE pour choisir u : Logs, Inverses trig, Algébrique, Trig, Expo

Produit xe^x

$$\int xe^x dx = (x-1)e^x$$

Utile pour $E(X)$ de la loi exponentielle

Produit x²e^x

$$\int x^2 e^x dx = (x^2-2x+2)e^x$$

Utile pour $E(X^2)$ et donc $V(X)$

Produit xe^{-λx}

$$\int xe^{-\lambda x} dx = -\frac{1}{\lambda^2}(\lambda x + 1)e^{-\lambda x}$$

Pour la loi exponentielle de paramètre λ

Bonus Cas Particuliers Utiles

Au moins un succès

$$P(X \geq 1) = 1 - P(X = 0) = 1 - (1-p)^n$$

Très utile pour les problèmes "au moins un"

Test médical (Bayes)

$$P(\text{Malade}|\text{+}) = \frac{P(+|\text{Malade}) \cdot P(\text{Malade})}{P(+)}$$

Attention au paradoxe du faux positif !

Tirage avec remise

$$\text{Nombre de tirages} = n^k$$

k tirages dans un ensemble de n éléments

Tirage sans remise

$$\text{Si ordre compte: } A_n^k \quad \text{sinon: } C_n^k$$

Choisir selon si l'ordre est important
