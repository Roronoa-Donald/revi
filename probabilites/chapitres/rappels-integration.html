1. Rappels sur les Fonctions

1.1 Qu'est-ce qu'une fonction ?

Une fonction est une relation qui associe à chaque valeur $x$ (l'entrée) une unique valeur $f(x)$ (la sortie).

$f : x \mapsto f(x)$

Exemple : $f(x) = x^2$ signifie "à chaque nombre $x$, on associe son carré"

Exemple concret

Si $f(x) = 2x + 3$, alors :

  • $f(0) = 2(0) + 3 = 3$
  • $f(1) = 2(1) + 3 = 5$
  • $f(5) = 2(5) + 3 = 13$

1.2 Les fonctions usuelles

📊 Fonctions polynômes

Ce sont les fonctions construites avec des puissances entières de $x$ :

  • Constante : $f(x) = c$ (exemple : $f(x) = 5$)
  • Affine : $f(x) = ax + b$ (exemple : $f(x) = 3x - 2$)
  • Quadratique : $f(x) = ax^2 + bx + c$ (exemple : $f(x) = x^2 - 4x + 3$)
  • Degré n : $f(x) = a_n x^n + a_{n-1}x^{n-1} + \ldots + a_0$

📊 Fonction racine carrée

$f(x) = \sqrt{x}$ définie pour $x \geq 0$

Notation alternative : $\sqrt{x} = x^{1/2}$

📊 Fonction exponentielle

$f(x) = e^x$ où $e \approx 2.718...$

Propriétés : $e^{a+b} = e^a \times e^b$ et $e^0 = 1$

L'exponentielle est fondamentale en probabilités (loi exponentielle, loi normale).

📊 Fonction logarithme népérien

$f(x) = \ln(x)$ définie pour $x > 0$

C'est la fonction réciproque de l'exponentielle : $\ln(e^x) = x$ et $e^{\ln(x)} = x$

Propriétés clés :

  • $\ln(ab) = \ln(a) + \ln(b)$
  • $\ln(a/b) = \ln(a) - \ln(b)$
  • $\ln(a^n) = n \ln(a)$
  • $\ln(1) = 0$ et $\ln(e) = 1$

🎯 Exercices - Fonctions

Exercice 1 Facile
Soit $f(x) = 3x - 5$. Calculez $f(4)$.
Remplacez $x$ par $4$ dans l'expression : $f(4) = 3 \times 4 - 5$
Exercice 2 Facile
Calculez $\sqrt{49}$.
Quel nombre multiplié par lui-même donne 49 ?
Exercice 3 Moyen
Simplifiez $e^2 \times e^3$ (répondez sous forme $e^n$, n étant un nombre).
Utilisez la propriété $e^a \times e^b = e^{a+b}$
Exercice 4 Moyen
Simplifiez $\ln(e^7)$.
Le logarithme et l'exponentielle sont des fonctions réciproques : $\ln(e^x) = x$
Exercice 5 Avancé
Simplifiez $\ln(8) - \ln(2)$ (donnez le résultat sous forme $\ln(n)$).
Utilisez $\ln(a) - \ln(b) = \ln(a/b)$, donc $\ln(8) - \ln(2) = \ln(8/2)$

2. La Dérivée

2.1 Qu'est-ce qu'une dérivée ?

La dérivée d'une fonction $f$ en un point $x$ mesure la vitesse de variation de $f$ en ce point. Géométriquement, c'est la pente de la tangente à la courbe.

$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$

On note $f'(x)$ ou $\frac{df}{dx}$ ou $\frac{d}{dx}f(x)$

Interprétation

  • Si $f'(x) > 0$ : la fonction croît en $x$
  • Si $f'(x) < 0$ : la fonction décroît en $x$
  • Si $f'(x) = 0$ : possible extremum (max ou min)

2.2 Formules de dérivation

Fonction $f(x)$ Dérivée $f'(x)$ Exemple
$c$ (constante) $0$ $(5)' = 0$
$x$ $1$ $(x)' = 1$
$x^n$ $n x^{n-1}$ $(x^3)' = 3x^2$
$\sqrt{x} = x^{1/2}$ $\frac{1}{2\sqrt{x}}$ $(\sqrt{x})' = \frac{1}{2\sqrt{x}}$
$\frac{1}{x} = x^{-1}$ $-\frac{1}{x^2}$ $(1/x)' = -1/x^2$
$e^x$ $e^x$ $(e^x)' = e^x$
$\ln(x)$ $\frac{1}{x}$ $(\ln x)' = 1/x$

2.3 Règles de dérivation

Somme : $(f + g)' = f' + g'$

Produit par constante : $(k \cdot f)' = k \cdot f'$

Produit : $(f \times g)' = f' \times g + f \times g'$

Quotient : $\left(\frac{f}{g}\right)' = \frac{f' \times g - f \times g'}{g^2}$

Composée : $(f \circ g)' = g' \times f'(g)$ ou $(f(g(x)))' = g'(x) \times f'(g(x))$

Exemple : Dériver $f(x) = 3x^2 + 5x - 2$

On applique les règles terme par terme :

  • $(3x^2)' = 3 \times 2x = 6x$
  • $(5x)' = 5$
  • $(-2)' = 0$

Résultat : $f'(x) = 6x + 5$

🎯 Exercices - Dérivées

Exercice 1 Facile
Dérivez $f(x) = x^4$. Donnez $f'(x)$.
Formule : $(x^n)' = nx^{n-1}$, ici $n=4$
Exercice 2 Facile
Dérivez $f(x) = 5e^x$. Donnez $f'(x)$.
$(e^x)' = e^x$ et $(kf)' = kf'$, donc $(5e^x)' = 5 \times (e^x)' = 5e^x$
Exercice 3 Moyen
Dérivez $f(x) = 2x^3 - 4x + 7$. Donnez $f'(x)$.
Dérivez terme par terme : $(2x^3)' = 6x^2$, $(-4x)' = -4$, $(7)' = 0$
Exercice 4 Moyen
Dérivez $f(x) = e^{2x}$ (fonction composée). Donnez $f'(x)$.
Utilisez la règle de la composée : si $g(x) = 2x$, alors $(e^{g(x)})' = g'(x) \times e^{g(x)} = 2 \times e^{2x}$
Exercice 5 Avancé
Dérivez $f(x) = x^2 e^x$ (produit). Donnez $f'(x)$ sous forme factorisée.
Règle du produit : $(uv)' = u'v + uv'$. Ici $u = x^2$, $v = e^x$.
$f'(x) = 2x \cdot e^x + x^2 \cdot e^x = e^x(2x + x^2) = e^x(x^2 + 2x)$

3. Primitives et Intégrales

3.1 Qu'est-ce qu'une primitive ?

Une primitive de $f$ est une fonction $F$ dont la dérivée est $f$ :

$F'(x) = f(x)$

On écrit $F(x) = \int f(x) \, dx$ (intégrale indéfinie)

Remarque importante : Une primitive n'est jamais unique ! Si $F$ est une primitive de $f$, alors $F + C$ (où $C$ est une constante) est aussi une primitive de $f$.

Exemple

On cherche une primitive de $f(x) = 2x$.

On sait que $(x^2)' = 2x$, donc $F(x) = x^2$ est une primitive de $f$.

Plus généralement : $F(x) = x^2 + C$ pour toute constante $C$.

3.2 L'intégrale définie

L'intégrale définie de $f$ entre $a$ et $b$ représente l'aire sous la courbe de $f$ entre ces bornes :

$$\int_a^b f(x) \, dx = F(b) - F(a)$$
où $F$ est une primitive quelconque de $f$

Exemple : Calculer $\int_0^2 3x^2 \, dx$

  1. Trouver une primitive de $3x^2$ : $F(x) = x^3$
  2. Calculer $F(2) - F(0) = 2^3 - 0^3 = 8 - 0 = 8$

Résultat : $\int_0^2 3x^2 \, dx = 8$

3.3 Formules de primitives

Fonction $f(x)$ Primitive $F(x) = \int f(x) dx$ Condition
$k$ (constante) $kx$ -
$x^n$ $\frac{x^{n+1}}{n+1}$ $n \neq -1$
$\frac{1}{x}$ $\ln|x|$ $x \neq 0$
$e^x$ $e^x$ -
$e^{ax}$ $\frac{1}{a}e^{ax}$ $a \neq 0$
$\sqrt{x} = x^{1/2}$ $\frac{2}{3}x^{3/2}$ $x > 0$
$\frac{1}{\sqrt{x}}$ $2\sqrt{x}$ $x > 0$

🎯 Exercices - Primitives et Intégrales

Exercice 1 Facile
Trouvez une primitive de $f(x) = 5$. (Sans la constante)
La primitive d'une constante $k$ est $kx$.
Exercice 2 Facile
Trouvez une primitive de $f(x) = x^3$. (Sans la constante)
Formule : $\int x^n dx = \frac{x^{n+1}}{n+1}$, ici $n=3$
Exercice 3 Moyen
Calculez $\int_0^1 2x \, dx$.
Primitive de $2x$ : $F(x) = x^2$. Calculez $F(1) - F(0) = 1 - 0$
Exercice 4 Moyen
Calculez $\int_0^1 e^x \, dx$. (Arrondissez à 2 décimales)
Primitive de $e^x$ : $F(x) = e^x$. Calculez $e^1 - e^0 = e - 1 \approx 2.718 - 1 = 1.718$
Exercice 5 Avancé
Calculez $\int_1^e \frac{1}{x} \, dx$.
Primitive de $\frac{1}{x}$ : $F(x) = \ln(x)$. Calculez $\ln(e) - \ln(1) = 1 - 0 = 1$

Fonction $f(x)$ Dérivée $f'(x)$ Primitive $\int f(x) dx$
$k$ (constante) $0$ $kx$
$x$ $1$ $\frac{x^2}{2}$
$x^n$ $nx^{n-1}$ $\frac{x^{n+1}}{n+1}$ $(n \neq -1)$
$\frac{1}{x}$ $-\frac{1}{x^2}$ $\ln|x|$
$\sqrt{x}$ $\frac{1}{2\sqrt{x}}$ $\frac{2}{3}x\sqrt{x}$

⚠️ Rappel essentiel pour les probabilités continues :

Pour une densité $f(x)$ : $\int_{-\infty}^{+\infty} f(x) dx = 1$

Pour une probabilité : $P(a \leq X \leq b) = \int_a^b f(x) dx$

5. Intégration par Parties (IPP)

5.1 La formule

L'intégration par parties est une technique essentielle pour calculer des intégrales de produits de fonctions.

$$\int u \cdot v' \, dx = u \cdot v - \int u' \cdot v \, dx$$

Ou avec les bornes : $$\int_a^b u \cdot v' \, dx = \left[u \cdot v\right]_a^b - \int_a^b u' \cdot v \, dx$$

5.2 Comment choisir $u$ et $v'$ ?

Règle LIATE (ordre de priorité pour choisir $u$) :

  1. Logarithmes : $\ln(x)$
  2. Inverses trigonométriques : $\arctan(x)$, etc.
  3. Algébriques (polynômes) : $x$, $x^2$, etc.
  4. Trigonométriques : $\sin(x)$, $\cos(x)$
  5. Exponentielles : $e^x$

On choisit $u$ le plus haut possible dans cette liste, et $v'$ le reste.

Exemple détaillé : $\int x e^x dx$

Étape 1 : Identifier $u$ et $v'$

  • $u = x$ (algébrique) → $u' = 1$
  • $v' = e^x$ (exponentielle) → $v = e^x$

Étape 2 : Appliquer la formule

$\int x e^x dx = x \cdot e^x - \int 1 \cdot e^x dx = xe^x - e^x + C$

Résultat : $\int x e^x dx = (x-1)e^x + C$

Exemple : $\int \ln(x) dx$

Astuce : On écrit $\ln(x) = 1 \times \ln(x)$

  • $u = \ln(x)$ → $u' = \frac{1}{x}$
  • $v' = 1$ → $v = x$

$\int \ln(x) dx = x\ln(x) - \int x \cdot \frac{1}{x} dx = x\ln(x) - \int 1 dx = x\ln(x) - x + C$

Exemple avancé : $\int x^2 e^x dx$ (double IPP)

1ère IPP :

  • $u = x^2$ → $u' = 2x$
  • $v' = e^x$ → $v = e^x$

$\int x^2 e^x dx = x^2 e^x - \int 2x e^x dx$

2ème IPP sur $\int 2x e^x dx$ :

$\int 2x e^x dx = 2(xe^x - e^x) = 2e^x(x-1)$

Résultat final :

$\int x^2 e^x dx = x^2 e^x - 2xe^x + 2e^x + C = e^x(x^2 - 2x + 2) + C$

🎯 Exercices - Intégration par Parties

Exercice 1 Facile
Dans $\int x \cos(x) dx$, que choisissez-vous pour $u$ ? (répondez "x" ou "cos(x)")
Règle LIATE : Algébrique (x) vient avant Trigonométrique (cos), donc on choisit $u = x$
Exercice 2 Facile
Calculez $\int_0^1 xe^x dx$. (Arrondissez à 2 décimales)
Primitive de $xe^x$ : $(x-1)e^x$.
$[(x-1)e^x]_0^1 = (1-1)e^1 - (0-1)e^0 = 0 - (-1) = 1$
Exercice 3 Moyen
Calculez $\int_1^e \ln(x) dx$. (Arrondissez à 2 décimales)
Primitive de $\ln(x)$ : $x\ln(x) - x$.
$[x\ln(x) - x]_1^e = (e \cdot 1 - e) - (1 \cdot 0 - 1) = (e - e) - (-1) = 0 + 1 = 1$
Exercice 4 Moyen
Trouvez la primitive de $x^2 e^x$ (sous forme factorisée avec $e^x$).
Double IPP nécessaire. Après calcul : $e^x(x^2 - 2x + 2)$
Exercice 5 Avancé
Calculez $\int_0^1 x^2 e^x dx$. (Arrondissez à 2 décimales)
Primitive : $e^x(x^2 - 2x + 2)$.
$[e^x(x^2-2x+2)]_0^1 = e^1(1-2+2) - e^0(0-0+2) = e \cdot 1 - 1 \cdot 2 = e - 2 \approx 0.718$

🔥 Exercices supplémentaires - Maîtriser l'IPP

Exercice 6 Moyen
Calculez $\int_0^2 x e^{-x} dx$. (Arrondissez à 3 décimales)
$u = x$, $v' = e^{-x}$ donc $v = -e^{-x}$.
Primitive de $xe^{-x}$ : $-(x+1)e^{-x}$.
$[-(x+1)e^{-x}]_0^2 = -3e^{-2} - (-1) = 1 - 3e^{-2} \approx 0.594$
Exercice 7 Avancé
Calculez $\int_0^{+\infty} x e^{-x} dx$ (intégrale impropre). Cette intégrale apparaît dans le calcul de $E(X)$ pour une loi exponentielle !
$\lim_{b \to +\infty} [-(x+1)e^{-x}]_0^b = \lim_{b \to +\infty} (-(b+1)e^{-b} + 1)$.
Comme $(b+1)e^{-b} \to 0$ quand $b \to +\infty$, on a : résultat = $1$
Exercice 8 Avancé
Calculez $\int_0^{+\infty} x^2 e^{-x} dx$. Cette intégrale sert à calculer $E(X^2)$ !
Double IPP. Primitive de $x^2 e^{-x}$ : $-(x^2+2x+2)e^{-x}$.
$\lim_{b \to +\infty} [-(x^2+2x+2)e^{-x}]_0^b = 0 - (-2) = 2$
Exercice 9 Avancé
Pour une loi exponentielle de paramètre $\lambda = 1$, la variance est $V(X) = E(X^2) - [E(X)]^2$. Avec les résultats précédents, calculez $V(X)$.
$E(X) = 1$ (exercice 7), $E(X^2) = 2$ (exercice 8).
$V(X) = 2 - 1^2 = 2 - 1 = 1$
Exercice 10 Avancé
Calculez $\int_0^1 x \ln(x) dx$. (Arrondissez à 2 décimales)
$u = \ln(x)$, $v' = x$. Donc $u' = 1/x$, $v = x^2/2$.
$\int x\ln(x)dx = \frac{x^2}{2}\ln(x) - \int \frac{x}{2}dx = \frac{x^2}{2}\ln(x) - \frac{x^2}{4}$
Note : $\lim_{x \to 0^+} x^2 \ln(x) = 0$. Calculez $[\frac{x^2}{2}\ln(x) - \frac{x^2}{4}]_0^1$

6. Exercices Récapitulatifs

Ces exercices mélangent tous les concepts vus. Ils sont du type de ceux que vous rencontrerez dans le chapitre 4 sur les variables continues.

Exercice Final 1 Moyen
Soit $f(x) = 2e^{-2x}$ pour $x \geq 0$. Vérifiez que c'est une densité de probabilité en calculant $\int_0^{+\infty} 2e^{-2x} dx$.
Primitive de $2e^{-2x}$ : $-e^{-2x}$.
$[-e^{-2x}]_0^{+\infty} = 0 - (-1) = 1$ ✓
Exercice Final 2 Moyen
Avec $f(x) = 2e^{-2x}$ pour $x \geq 0$, calculez $P(X \leq 1) = \int_0^1 2e^{-2x} dx$. (Arrondissez à 3 décimales)
$[-e^{-2x}]_0^1 = -e^{-2} - (-1) = 1 - e^{-2} \approx 1 - 0.135 = 0.865$
Exercice Final 3 Avancé
Calculez l'espérance $E(X) = \int_0^{+\infty} x \cdot 2e^{-2x} dx$. (Réponse exacte en fraction)
C'est une loi exponentielle de paramètre $\lambda = 2$.
$E(X) = 1/\lambda = 1/2$ pour une loi exponentielle.
(Ou calculez par IPP avec $u = x$, $v' = 2e^{-2x}$)
Exercice Final 4 Avancé
Soit $f(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2/2}$ (densité de la loi normale centrée réduite).
Que vaut $\int_{-\infty}^{+\infty} f(x) dx$ ? (C'est un résultat classique à connaître)
Toute densité de probabilité vérifie $\int_{-\infty}^{+\infty} f(x) dx = 1$ par définition !
Exercice Final 5 Avancé
La fonction de répartition $F(x) = \int_{-\infty}^{x} f(t) dt$. Si $f(t) = e^{-t}$ pour $t \geq 0$ (et 0 sinon), trouvez $F(x)$ pour $x \geq 0$.
$F(x) = \int_0^x e^{-t} dt = [-e^{-t}]_0^x = -e^{-x} - (-1) = 1 - e^{-x}$
