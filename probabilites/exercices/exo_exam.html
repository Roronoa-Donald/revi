Fonction de répartition donnée — $F_X$

CDF · Densité · Probabilités ponctuelles · Variable continue

Soit $X$ une variable aléatoire dont la fonction de répartition est donnée par :

$$F_X(x) = \begin{cases} 0 & \text{si } x < 0 \\ \dfrac{x}{2} & \text{si } x \in [0,\, 1[ \\ 1 & \text{si } x \geq 1 \end{cases}$$
  1. Tracer le graphe de $F_X$. Est-ce que $X$ est une variable discrète ? Une variable à densité ?
  2. Calculer :
    1. $P(X = \tfrac{1}{2})$
    2. $P(X = 1)$
    3. $P(\tfrac{3}{4} < X \leq 1)$
    4. $P(\tfrac{3}{4} \leq X \leq 1)$
1
Nature de $X$ : discrète ou à densité ?
Analysez la forme de $F_X$. Est-elle en escalier ? Est-elle continue et dérivable par morceaux ?
Indice

Une variable discrète a une CDF en escalier (sauts). Une variable à densité a une CDF continue.

Ici, $F_X$ est-elle continue ? Vérifiez en $x = 0$ et $x = 1$.

Correction détaillée
Graphe de $F_X$ :
• Pour $x < 0$ : $F_X(x) = 0$ (droite horizontale sur l'axe des $x$)
• Pour $x \in [0, 1[$ : $F_X(x) = \frac{x}{2}$ (segment de droite de $(0,0)$ à $(1, \frac{1}{2})$)
• Pour $x \geq 1$ : $F_X(x) = 1$ (droite horizontale à hauteur 1)
Continuité :
• En $x = 0$ : $\lim_{x \to 0^-} F_X(x) = 0 = F_X(0)$ ✓ continue
• En $x = 1$ : $\lim_{x \to 1^-} F_X(x) = \frac{1}{2} \neq 1 = F_X(1)$ ✗ discontinue !
$F_X$ n'est pas continue en $x=1$ (saut de $\frac{1}{2}$), donc $X$ n'est pas à densité.
$F_X$ n'est pas non plus en escalier pur (elle n'a pas que des sauts), donc $X$ n'est pas discrète non plus.

Conclusion : $X$ est une variable aléatoire mixte — elle a une partie continue (sur $[0,1[$) et un atome (un saut) en $x = 1$.
2a
Calculer $P(X = \tfrac{1}{2})$
$P(X = a) = F_X(a) - F_X(a^-)$ (le saut de $F$ en $a$). Que vaut le saut en $x = \frac{1}{2}$ ?
Indice

$F_X$ est continue en $x = \frac{1}{2}$ (pas de saut). Pour toute variable continue en un point : $P(X=a) = 0$.

Correction
$F_X$ est continue en $x = \frac{1}{2}$ (pas de saut) donc : $$P\!\left(X = \tfrac{1}{2}\right) = F_X\!\left(\tfrac{1}{2}\right) - \lim_{x \to \frac{1}{2}^-} F_X(x) = \frac{1}{4} - \frac{1}{4} = \boxed{0}$$
2b
Calculer $P(X = 1)$
$P(X = 1) = F_X(1) - F_X(1^-)$. Quel est le saut de $F_X$ en $x=1$ ?
Indice

$F_X(1) = 1$ et $\lim_{x \to 1^-} F_X(x) = \lim_{x \to 1^-} \frac{x}{2} = \frac{1}{2}$. Le saut est $1 - \frac{1}{2}$.

Correction
$$P(X = 1) = F_X(1) - \lim_{x \to 1^-} F_X(x) = 1 - \frac{1}{2} = \boxed{\frac{1}{2}}$$ $X$ a un atome en $x = 1$ : la probabilité $P(X=1) = \frac{1}{2}$ est concentrée en ce point.
2c
Calculer $P(\tfrac{3}{4} < X \leq 1)$
Utilisez $P(a < X \leq b) = F_X(b) - F_X(a)$.
Indice

$P(\frac{3}{4} < X \leq 1) = F_X(1) - F_X(\frac{3}{4}) = 1 - \frac{3}{8}$

Correction
$$P\!\left(\frac{3}{4} < X \leq 1\right) = F_X(1) - F_X\!\left(\frac{3}{4}\right) = 1 - \frac{3/4}{2} = 1 - \frac{3}{8} = \boxed{\frac{5}{8}}$$
2d
Calculer $P(\tfrac{3}{4} \leq X \leq 1)$
Attention : $P(a \leq X \leq b) = P(a < X \leq b) + P(X = a)$. Quelle différence avec la question précédente ?
Indice

$P(\frac{3}{4} \leq X \leq 1) = P(\frac{3}{4} < X \leq 1) + P(X = \frac{3}{4})$. Or $F_X$ est continue en $x = \frac{3}{4}$, donc $P(X=\frac{3}{4}) = 0$.

Correction
$$P\!\left(\frac{3}{4} \leq X \leq 1\right) = P\!\left(\frac{3}{4} < X \leq 1\right) + P\!\left(X = \frac{3}{4}\right)$$ $F_X$ est continue en $x = \frac{3}{4}$ (pas de saut), donc $P(X = \frac{3}{4}) = 0$. $$= \frac{5}{8} + 0 = \boxed{\frac{5}{8}}$$ Ici, $\leq$ ou $<$ en borne inférieure ne change rien car il n'y a pas d'atome en $\frac{3}{4}$.

Corrigé complet — exercice 1

1) Nature de $X$

$F_X$ est continue partout sauf en $x = 1$ où elle présente un saut de $\Delta = 1 - \frac{1}{2} = \frac{1}{2}$.
$X$ n'est ni discrète (pas en escalier) ni à densité (pas continue). C'est une variable mixte.
Sur $[0,1[$, $X$ admet la densité $f(x) = F_X'(x) = \frac{1}{2}$, et en $x = 1$ il y a un atome de masse $\frac{1}{2}$.

2) Calculs de probabilités

a) $P(X = \frac{1}{2}) = \boxed{0}$ (pas de saut en $\frac{1}{2}$)
b) $P(X = 1) = 1 - \frac{1}{2} = \boxed{\frac{1}{2}}$ (saut en $x=1$)
c) $P(\frac{3}{4} < X \leq 1) = F_X(1) - F_X(\frac{3}{4}) = 1 - \frac{3}{8} = \boxed{\frac{5}{8}}$
d) $P(\frac{3}{4} \leq X \leq 1) = \frac{5}{8} + P(X=\frac{3}{4}) = \frac{5}{8} + 0 = \boxed{\frac{5}{8}}$

Densité affine — $k(x+2)$ sur $[-2,\,0]$

Densité · CDF · Probabilités · Espérance · Variance

On donne :

$$f(x) = \begin{cases} k(x+2) & \text{si } -2 \leq x \leq 0 \\ 0 & \text{sinon} \end{cases}$$
  1. Déterminer la constante $k$ pour que $f$ soit une densité.
  2. Déterminer $F(x)$ puis calculer :
    1. $P(X > 1)$
    2. $P(X < -1)$
    3. $P(X > -\tfrac{1}{2})$
    4. $P(-4 < X < 0)$
    5. $P(-1 \leq X \leq -\tfrac{1}{2})$
  3. Calculer $E(X)$, $V(X)$ et $\sigma(X)$.
1
Déterminer $k$
Condition de normalisation : $\displaystyle\int_{-2}^{0} k(x+2)\,dx = 1$. Résolvez pour $k$.
Indice

$\int_{-2}^{0} k(x+2)\,dx = k\left[\frac{(x+2)^2}{2}\right]_{-2}^{0} = k \times \frac{4}{2} = 2k = 1$

Correction
$$\int_{-2}^{0} k(x+2)\,dx = k\left[\frac{x^2}{2} + 2u\right]_{-2}^{0} = k\left[0 - (2-4)\right] = 2k = 1$$ $$\boxed{k = \frac{1}{2}}$$ Donc $f(x) = \dfrac{x+2}{2}$ sur $[-2,\, 0]$.
2
Déterminer $F(x)$
Calculez $F(x) = \int_{-\infty}^{x} f(t)\,dt$ pour les trois cas : $x < -2$, $-2 \leq x \leq 0$ et $x > 0$.
Indice

Pour $-2 \leq x \leq 0$ :

$$F(x) = \int_{-2}^{x} \frac{t+2}{2}\,dt = \frac{1}{2}\left[\frac{(t+2)^2}{2}\right]_{-2}^{x} = \frac{(x+2)^2}{4}$$

Correction
$$\boxed{F(x) = \begin{cases} 0 & \text{si } x < -2 \\ \dfrac{(x+2)^2}{4} & \text{si } -2 \leq x \leq 0 \\ 1 & \text{si } x > 0 \end{cases}}$$ Vérification : $F(-2) = 0$ ✓ et $F(0) = 4/4 = 1$ ✓.
2a
Calculer $P(X > 1)$
Observez le support de $X$. Quelles valeurs peut prendre $X$ ?
Indice

Le support de $X$ est $[-2,\, 0]$. $X$ ne peut pas dépasser 0.

Correction
$X \in [-2,\, 0]$ donc $P(X > 1) = \boxed{0}$ (événement impossible).
2b
Calculer $P(X < -1)$
$P(X < -1) = F(-1)$.
Indice

$F(-1) = \frac{(-1+2)^2}{4} = \frac{1}{4}$

Correction
$$P(X < -1) = F(-1) = \frac{(-1+2)^2}{4} = \frac{1}{4} = \boxed{0{,}25}$$
2c
Calculer $P(X > -\tfrac{1}{2})$
$P(X > -1/2) = 1 - F(-1/2)$.
Indice

$F(-1/2) = \frac{(-1/2+2)^2}{4} = \frac{(3/2)^2}{4} = \frac{9}{16}$

Correction
$$P(X > -\tfrac{1}{2}) = 1 - F(-\tfrac{1}{2}) = 1 - \frac{9}{16} = \boxed{\frac{7}{16} \approx 0{,}4375}$$
2d
Calculer $P(-4 < X < 0)$
Comparez l'intervalle $]-4,\, 0[$ avec le support de $X$.
Indice

Le support $[-2,\, 0] \subset ]-4,\, 0[$, donc l'intervalle contient tout le support.

Correction
$[-2,\, 0] \subset ]-4,\, 0[$ donc $P(-4 < X < 0) = \boxed{1}$ (événement certain).
2e
Calculer $P(-1 \leq X \leq -\tfrac{1}{2})$
$P(-1 \leq X \leq -1/2) = F(-1/2) - F(-1)$.
Indice

$F(-1/2) - F(-1) = \frac{9}{16} - \frac{1}{4} = \frac{9}{16} - \frac{4}{16} = \frac{5}{16}$

Correction
$$P(-1 \leq X \leq -\tfrac{1}{2}) = F(-\tfrac{1}{2}) - F(-1) = \frac{9}{16} - \frac{1}{4} = \frac{9-4}{16} = \boxed{\frac{5}{16} \approx 0{,}3125}$$
3
Calculer $E(X)$, $V(X)$ et $\sigma(X)$
Calculez $E(X) = \int_{-2}^{0} x \cdot f(x)\,dx$, puis $E(X^2)$ et déduisez $V(X)$ et $\sigma(X)$.
Indice

$E(X) = \int_{-2}^{0} x \cdot \frac{x+2}{2}\,dx = \frac{1}{2}\int_{-2}^{0}(x^2+2x)\,dx$

$E(X^2) = \frac{1}{2}\int_{-2}^{0}(x^3+2x^2)\,dx$, puis $V(X) = E(X^2) - [E(X)]^2$.

Correction détaillée
$E(X)$ : $$E(X) = \frac{1}{2}\int_{-2}^{0}(x^2+2x)\,dx = \frac{1}{2}\left[\frac{x^3}{3}+x^2\right]_{-2}^{0} = \frac{1}{2}\left[0 - \left(-\frac{8}{3}+4\right)\right] = \frac{1}{2} \times \left(-\frac{4}{3}\right) = \boxed{-\frac{2}{3}}$$
$E(X^2)$ : $$E(X^2) = \frac{1}{2}\int_{-2}^{0}(x^3+2x^2)\,dx = \frac{1}{2}\left[\frac{x^4}{4}+\frac{2u^3}{3}\right]_{-2}^{0} = \frac{1}{2}\left[0 - \left(4 - \frac{16}{3}\right)\right] = \frac{1}{2} \times \frac{4}{3} = \boxed{\frac{2}{3}}$$
Variance et écart-type : $$V(X) = E(X^2) - [E(X)]^2 = \frac{2}{3} - \frac{4}{9} = \frac{6-4}{9} = \boxed{\frac{2}{9}}$$ $$\sigma(X) = \sqrt{\frac{2}{9}} = \frac{\sqrt{2}}{3} \approx \boxed{0{,}471}$$

Corrigé complet — exercice 2

1) Constante $k$

$\int_{-2}^{0} k(x+2)du = 2k = 1 \Rightarrow \boxed{k = 1/2}$. Donc $f(x) = (x+2)/2$.

2) CDF et probabilités

$F(x) = \frac{(x+2)^2}{4}$ pour $-2 \leq x \leq 0$, $0$ avant, $1$ après.
a) $P(X > 1) = \boxed{0}$ (support $[-2, 0]$)
b) $P(X < -1) = F(-1) = 1/4 = \boxed{0{,}25}$
c) $P(X > -1/2) = 1 - 9/16 = \boxed{7/16}$
d) $P(-4 < X < 0) = \boxed{1}$ (support $\subset$ intervalle)
e) $P(-1 \leq X \leq -1/2) = 9/16 - 1/4 = \boxed{5/16}$

3) Espérance, Variance, Écart-type

$$\boxed{E(X) = -\frac{2}{3}} \qquad \boxed{V(X) = \frac{2}{9}} \qquad \boxed{\sigma(X) = \frac{\sqrt{2}}{3} \approx 0{,}471}$$

Loi exponentielle — $3\,e^{-3x}$

Loi exponentielle · Densité · CDF · Probabilités · Espérance · Variance

On considère la fonction :

$$f(x) = \begin{cases} 3\,e^{-3x} & \text{si } x > 0 \\ 0 & \text{sinon} \end{cases}$$
  1. Montrer que $f$ est une fonction de densité de probabilité.
  2. Déterminer la fonction de répartition $F(x)$, puis calculer :
    1. $P(X > 0)$
    2. $P(|X| \leq \tfrac{1}{3})$
    3. $P(0{,}5 \leq X \leq 30)$
    4. $P(X < 3)$
    5. $P(-1 < X < 1)$
    6. $P\!\left(X < \tfrac{4}{3} \mid X > \tfrac{1}{3}\right)$
  3. Calculer $E(X)$, $V(X)$ et $\sigma(X)$.
1
Montrer que $f$ est une densité de probabilité
Vérifiez la positivité et que l'intégrale sur $\mathbb{R}$ vaut 1.
Indice

$f(x) \geq 0$ car $e^{-3x} > 0$. Pour l'intégrale :

$$\int_0^{+\infty} 3\,e^{-3x}\,dx = 3 \times \frac{1}{3} = 1$$

Correction détaillée
Positivité : $f(x) = 0$ pour $x \leq 0$ ✓ ; $f(x) = 3e^{-3x} > 0$ pour $x > 0$ ✓
Normalisation : $$\int_{-\infty}^{+\infty} f(x)\,dx = \int_0^{+\infty} 3\,e^{-3x}\,dx = 3\left[-\frac{1}{3}e^{-3x}\right]_0^{+\infty} = 3 \times \frac{1}{3} = \boxed{1} \ ✓$$
Conclusion : $f$ est bien une densité. On reconnaît la loi exponentielle $\mathcal{E}(\lambda = 3)$.
2
Déterminer la fonction de répartition $F(x)$
Calculez $F(x) = \int_{-\infty}^{x} f(t)\,dt$ pour $x \leq 0$ et $x > 0$.
Indice

Pour $x > 0$ : $F(x) = \int_0^x 3e^{-3t}dt = [-e^{-3t}]_0^x = 1 - e^{-3x}$

Correction
$$\boxed{F(x) = \begin{cases} 0 & \text{si } x \leq 0 \\ 1 - e^{-3x} & \text{si } x > 0 \end{cases}}$$
2a
Calculer $P(X > 0)$
$P(X > 0) = 1 - F(0)$. Que vaut $F(0)$ ?
Indice

$F(0) = 0$ (limite gauche), donc $P(X > 0) = 1 - 0 = 1$. Toute la masse est sur $]0, +\infty[$.

Correction
$$P(X > 0) = 1 - F(0) = 1 - 0 = \boxed{1}$$ Le support de $X$ est $]0, +\infty[$ : $X$ est presque sûrement positive.
2b
Calculer $P(|X| \leq \tfrac{1}{3})$
$|X| \leq \frac{1}{3}$ signifie $-\frac{1}{3} \leq X \leq \frac{1}{3}$. Mais $X \geq 0$ p.s., donc…
Indice

Comme $X \geq 0$ : $P(|X| \leq 1/3) = P(0 \leq X \leq 1/3) = F(1/3)$.

Correction
Comme $X \geq 0$ p.s. : $P(|X| \leq \tfrac{1}{3}) = P(0 \leq X \leq \tfrac{1}{3}) = F\!\left(\tfrac{1}{3}\right)$ $$= 1 - e^{-3 \times 1/3} = 1 - e^{-1} \approx \boxed{0{,}632}$$
2c
Calculer $P(0{,}5 \leq X \leq 30)$
$P(a \leq X \leq b) = F(b) - F(a)$.
Indice

$F(30) - F(0{,}5) = (1 - e^{-90}) - (1 - e^{-3/2}) = e^{-3/2} - e^{-90}$. Or $e^{-90} \approx 0$.

Correction
$$P(0{,}5 \leq X \leq 30) = F(30) - F(0{,}5) = e^{-3/2} - e^{-90}$$ $$\approx e^{-1{,}5} \approx \boxed{0{,}223}$$ $e^{-90}$ est négligeable. Presque toute la probabilité résiduelle au-delà de $0{,}5$ est concentrée près de cette borne.
2d
Calculer $P(X < 3)$
Appliquez directement $F(3)$.
Indice

$F(3) = 1 - e^{-9}$. Avec $\lambda = 3$, la valeur 3 est à $9$ écart-types de 0 !

Correction
$$P(X < 3) = F(3) = 1 - e^{-9} \approx \boxed{0{,}99988}$$ Quasi certain ! Avec $E(X) = 1/3$, la valeur $x=3$ est très loin dans la queue de distribution.
2e
Calculer $P(-1 < X < 1)$
Attention au support de $X$ ! Que vaut $P(X < 0)$ ?
Indice

$X \geq 0$ p.s., donc $P(-1 < X < 1) = P(0 \leq X < 1) = F(1) = 1 - e^{-3}$

Correction
Comme $X \geq 0$ : $P(-1 < X < 1) = P(0 \leq X < 1) = F(1)$ $$= 1 - e^{-3} \approx \boxed{0{,}950}$$
2f
Calculer $P\!\left(X < \tfrac{4}{3} \mid X > \tfrac{1}{3}\right)$
Probabilité conditionnelle. Utilisez la propriété sans mémoire pour simplifier.
Indice

$P(X < \frac{4}{3} \mid X > \frac{1}{3}) = 1 - P(X \geq \frac{4}{3} \mid X > \frac{1}{3})$

Par la propriété sans mémoire : $P(X > \frac{4}{3} \mid X > \frac{1}{3}) = P(X > \frac{4}{3} - \frac{1}{3}) = P(X > 1)$

Correction détaillée
Méthode 1 — Propriété sans mémoire : $$P\!\left(X \geq \frac{4}{3} \mid X > \frac{1}{3}\right) = P\!\left(X > \frac{4}{3} - \frac{1}{3}\right) = P(X > 1) = e^{-3}$$ $$P\!\left(X < \frac{4}{3} \mid X > \frac{1}{3}\right) = 1 - e^{-3} \approx \boxed{0{,}950}$$
Méthode 2 — Calcul direct : $$= \frac{P(\frac{1}{3} < X < \frac{4}{3})}{P(X > \frac{1}{3})} = \frac{F(4/3) - F(1/3)}{1 - F(1/3)} = \frac{e^{-1} - e^{-4}}{e^{-1}} = 1 - e^{-3} \approx 0{,}950$$
Remarque : C'est la même valeur que $P(X < 1) = F(1) = 1 - e^{-3}$ ! La propriété sans mémoire « remet le compteur à zéro ».
3
Calculer $E(X)$, $V(X)$ et $\sigma(X)$
Utilisez les formules de la loi exponentielle $\mathcal{E}(\lambda)$ ou calculez par intégration.
Indice

Pour $\mathcal{E}(\lambda)$ : $E(X) = \frac{1}{\lambda}$, $V(X) = \frac{1}{\lambda^2}$, $\sigma(X) = \frac{1}{\lambda}$.

Ici $\lambda = 3$.

Correction
Pour $\mathcal{E}(\lambda = 3)$ : $$\boxed{E(X) = \frac{1}{3}} \qquad \boxed{V(X) = \frac{1}{9}} \qquad \boxed{\sigma(X) = \frac{1}{3}}$$

Corrigé complet — exercice 3

1) $f$ est une densité

$f(x) \geq 0$ ✓ et $\int_0^{+\infty} 3e^{-3x}dx = 1$ ✓. C'est $\mathcal{E}(3)$.

2) CDF et probabilités

$F(x) = 1 - e^{-3x}$ pour $x > 0$, $0$ sinon.
a) $P(X > 0) = \boxed{1}$
b) $P(|X| \leq 1/3) = F(1/3) = 1 - e^{-1} \approx \boxed{0{,}632}$
c) $P(0{,}5 \leq X \leq 30) = e^{-3/2} - e^{-90} \approx \boxed{0{,}223}$
d) $P(X < 3) = 1 - e^{-9} \approx \boxed{0{,}99988}$
e) $P(-1 < X < 1) = F(1) = 1 - e^{-3} \approx \boxed{0{,}950}$
f) $P(X < 4/3 \mid X > 1/3) = 1 - e^{-3} \approx \boxed{0{,}950}$ (sans mémoire)

3) Espérance, Variance, Écart-type

$$\boxed{E(X) = \frac{1}{3}} \qquad \boxed{V(X) = \frac{1}{9}} \qquad \boxed{\sigma(X) = \frac{1}{3}}$$
4

Loi exponentielle — $\frac{1}{2}e^{-\frac{1}{2}x}$

Loi exponentielle · CDF · Propriété sans mémoire · Espérance · Variance

On considère la fonction :

$$f(x) = \begin{cases} \dfrac{1}{2}\,e^{-\frac{1}{2}x} & \text{si } x > 0 \\ 0 & \text{sinon} \end{cases}$$
  1. Montrer que $f$ est une fonction de densité de probabilité.
  2. Déterminer la fonction de répartition $F(x)$, puis calculer :
    1. $P(X > 4)$
    2. $P(X < 20)$
    3. $P(0{,}5 < X < 15)$
    4. $P(X < -30)$
    5. $P(X < 30 \mid X > 29{,}5)$
    6. $P(X > t + x \mid X > x)$ — propriété sans mémoire
  3. Calculer $E(X)$, $V(X)$ et $\sigma(X)$.
1
Montrer que $f$ est une densité de probabilité
Quelles sont les deux conditions pour qu'une fonction soit une densité ? Vérifiez-les pour $f$.
Indice

Deux conditions :

  1. $f(x) \geq 0$ pour tout $x$ (positivité)
  2. $\displaystyle\int_{-\infty}^{+\infty} f(x)\,dx = 1$ (normalisation)

Pour la positivité : la fonction exponentielle est toujours positive.

Pour l'intégrale : calculez $\displaystyle\int_0^{+\infty} \frac{1}{2}e^{-x/2}\,dx$. Rappel : $\displaystyle\int_0^{+\infty} e^{-\alpha x}\,dx = \frac{1}{\alpha}$.

Correction détaillée
Positivité
• Si $x \leq 0$ : $f(x) = 0 \geq 0$ ✓
• Si $x > 0$ : $e^{-x/2} > 0$ et $\frac{1}{2} > 0$, donc $f(x) = \frac{1}{2}e^{-x/2} > 0$ ✓
Normalisation
$$\int_{-\infty}^{+\infty} f(x)\,dx = \int_0^{+\infty} \frac{1}{2}e^{-x/2}\,dx = \frac{1}{2}\left[-2\,e^{-x/2}\right]_0^{+\infty}$$ $$= \frac{1}{2}\left(0 - (-2)\right) = \frac{1}{2} \times 2 = \boxed{1} \quad ✓$$
Conclusion : $f(x) \geq 0$ et $\int f = 1$, donc $f$ est bien une densité de probabilité.
On reconnaît la loi exponentielle $\mathcal{E}(\lambda)$ avec $\lambda = \frac{1}{2}$.
2
Déterminer la fonction de répartition $F(x)$
Calculez $F(x) = \displaystyle\int_{-\infty}^{x} f(t)\,dt$ pour $x \leq 0$ et $x > 0$.
Indice

Pour $x \leq 0$ : $f(t) = 0$ sur $(-\infty, x]$, donc $F(x) = 0$.

Pour $x > 0$ :

$$F(x) = \int_0^x \frac{1}{2}e^{-t/2}\,dt$$

La primitive de $e^{-t/2}$ est $-2e^{-t/2}$.

Correction détaillée
Cas $x \leq 0$ : $F(x) = 0$
Cas $x > 0$ : $$F(x) = \int_0^x \frac{1}{2}e^{-t/2}\,dt = \frac{1}{2}\left[-2\,e^{-t/2}\right]_0^x = -e^{-x/2} + 1$$
Résumé : $$\boxed{F(x) = \begin{cases} 0 & \text{si } x \leq 0 \\ 1 - e^{-x/2} & \text{si } x > 0 \end{cases}}$$
2a
Calculer $P(X > 4)$
Utilisez $F$ pour calculer $P(X > 4) = 1 - F(4)$.
Indice

$P(X > 4) = 1 - F(4) = 1 - (1 - e^{-4/2}) = e^{-2}$

Correction

Indice

Correction

$X_d \sim \mathcal{B}(n,\, e^{-d/82})$ — Loi binomiale
L'entreprise possède $n$ autocars indépendants. $X_d$ = nombre d'autocars sans incident après $d$ km.
Quelle est la loi de $X_d$ ? Quel est le nombre moyen d'autocars sans incident ?
Indice

Chaque autocar a une probabilité $p = P(D > d) = e^{-d/82}$ de ne pas avoir d'incident en $d$ km.

Les $n$ autocars sont indépendants → somme de $n$ Bernoulli indépendantes = loi binomiale.

Correction détaillée
Pour chaque autocar $i$, notons $Y_i = \mathbb{1}_{\{D_i > d\}}$ (1 si pas d'incident en $d$ km, 0 sinon). $$P(Y_i = 1) = P(D_i > d) = e^{-d/82} = p$$
Les $Y_i$ sont indépendantes et de même loi de Bernoulli $\mathcal{B}(p)$. Donc : $$X_d = \sum_{i=1}^n Y_i \sim \boxed{\mathcal{B}(n,\, e^{-d/82})}$$
Nombre moyen : $$E(X_d) = n \cdot p = \boxed{n\,e^{-d/82}}$$ Exemple : si $n=100$ autocars et $d=50$ km : $E(X_{50}) = 100 \times e^{-50/82} \approx 54{,}3$ autocars sans incident en moyenne.

Densité de probabilité — $(1-x)^{1/3}$

VA Continue · Densité · CDF · Espérance · Variance

On considère la fonction :

$$f(x) = \begin{cases} k\,(1-x)^{1/3} & \text{si } 0 \leq x \leq 1 \\ 0 & \text{sinon} \end{cases}$$
  1. Déterminer la constante $k$ pour que $f$ soit une densité de probabilité.
  2. En déduire la fonction de répartition $F(x)$, puis calculer :
    1. $P(X \leq 1)$
    2. $P\!\left(X > \tfrac{1}{4}\right)$
    3. $P\!\left(\tfrac{1}{4} < X < \tfrac{1}{2}\right)$
    4. $P(X > 3)$
    5. $P\!\left(X < 0{,}75 \mid X > 0{,}25\right)$
  3. Calculer $E(X)$ et $V(X)$.
1
Déterminer la constante $k$
Pour que $f$ soit une densité de probabilité, quelles conditions doit-elle vérifier ? Utilisez ces conditions pour trouver $k$.
Indice

Pour qu'une fonction soit une densité, elle doit satisfaire deux conditions :

  1. $f(x) \geq 0$ pour tout $x$ (positivité)
  2. $\displaystyle\int_{-\infty}^{+\infty} f(x)\,dx = 1$ (normalisation)

Ici $f$ est nulle hors de $[0,1]$, donc il suffit de calculer $\displaystyle\int_0^1 k(1-x)^{1/3}\,dx = 1$.

Astuce : posez le changement de variable $u = 1 - x$.

Correction détaillée
Étape 1 — Positivité
Pour $0 \leq x \leq 1$ : on a $0 \leq 1-x \leq 1$, donc $(1-x)^{1/3} \geq 0$.
Si $k > 0$, alors $f(x) = k(1-x)^{1/3} \geq 0$. ✓
Étape 2 — Normalisation
On doit résoudre : $$\int_0^1 k(1-x)^{1/3}\,dx = 1$$ Posons $u = 1-x$, donc $du = -dx$. Quand $x=0$, $u=1$ ; quand $x=1$, $u=0$. $$k\int_1^0 u^{1/3}(-du) = k\int_0^1 u^{1/3}\,du$$
Étape 3 — Calcul de l'intégrale
$$k\int_0^1 u^{1/3}\,du = k\left[\frac{u^{4/3}}{4/3}\right]_0^1 = k \cdot \frac{3}{4}\left[u^{4/3}\right]_0^1 = k \cdot \frac{3}{4}$$
Conclusion
$$k \cdot \frac{3}{4} = 1 \implies \boxed{k = \frac{4}{3}}$$
2
Déterminer la fonction de répartition $F(x)$
La fonction de répartition est définie par $F(x) = P(X \leq x) = \displaystyle\int_{-\infty}^{x} f(t)\,dt$.
Calculez $F(x)$ pour les trois cas : $x < 0$, $0 \leq x \leq 1$, et $x > 1$.
Indice

Comme $f(t) = 0$ pour $t < 0$ : $F(x) = 0$ pour $x < 0$.

Pour $0 \leq x \leq 1$ : $F(x) = \displaystyle\int_0^x \frac{4}{3}(1-t)^{1/3}\,dt$.

Utilisez le même changement de variable $u = 1-t$.

Pour $x > 1$ : $F(x) = 1$ (toute la probabilité est déjà atteinte).

Correction détaillée
Cas 1 : $x < 0$
$f(t) = 0$ pour tout $t < 0$, donc $F(x) = 0$.
Cas 2 : $0 \leq x \leq 1$
$$F(x) = \int_0^x \frac{4}{3}(1-t)^{1/3}\,dt$$ Posons $u = 1-t$, $du = -dt$ : $$= \frac{4}{3}\int_1^{1-x} u^{1/3}(-du) = \frac{4}{3}\int_{1-x}^1 u^{1/3}\,du$$ $$= \frac{4}{3} \cdot \frac{3}{4}\left[u^{4/3}\right]_{1-x}^1 = 1 - (1-x)^{4/3}$$
Cas 3 : $x > 1$
$F(x) = 1$ car tout l'intervalle de définition est couvert.
Résumé
$$\boxed{F(x) = \begin{cases} 0 & \text{si } x < 0 \\ 1 - (1-x)^{4/3} & \text{si } 0 \leq x \leq 1 \\ 1 & \text{si } x > 1 \end{cases}}$$
2a
Calculer $P(X \leq 1)$
Utilisez $F(x)$ pour calculer $P(X \leq 1)$. C'est immédiat !
Indice

$P(X \leq 1) = F(1)$. Remplacez $x$ par $1$ dans la formule de $F$.

Correction
$P(X \leq 1) = F(1) = 1 - (1-1)^{4/3} = 1 - 0 = \boxed{1}$
Ce résultat est logique : $X$ prend ses valeurs dans $[0,1]$, donc $P(X \leq 1) = 1$.
2b
Calculer $P\!\left(X > \tfrac{1}{4}\right)$
Exprimez $P(X > \frac{1}{4})$ à l'aide de $F$, puis calculez la valeur numérique.
Indice

$P(X > a) = 1 - P(X \leq a) = 1 - F(a)$.

Ici $a = \frac{1}{4}$, donc : $P(X > \frac{1}{4}) = 1 - F\!\left(\frac{1}{4}\right) = 1 - \left[1 - \left(\frac{3}{4}\right)^{4/3}\right]$.

Correction

2c
Calculer $P\!\left(\tfrac{1}{4} < X < \tfrac{1}{2}\right)$

Calculez chaque terme séparément :

  • $F\!\left(\frac{1}{2}\right) = 1 - \left(\frac{1}{2}\right)^{4/3}$

Calculer $P(X > 3)$
Regardez le support de la variable $X$ (l'ensemble des valeurs possibles). Que vaut cette probabilité ?
Indice

La variable $X$ ne prend des valeurs que dans $[0, 1]$. Que peut-on dire de $P(X > 3)$ ?

$$\boxed{P(X > 3) = 0}$$
2e
Calculer $P\!\left(X < 0{,}75 \mid X > 0{,}25\right)$
C'est une probabilité conditionnelle. Rappelez la formule de Bayes et simplifiez.
Indice

Formule de la probabilité conditionnelle :

$$P(A \mid B) = \frac{P(A \cap B)}{P(B)}$$

Ici $A = \{X < 0{,}75\}$ et $B = \{X > 0{,}25\}$.

$A \cap B = \{0{,}25 < X < 0{,}75\}$, donc :

$$P(A \mid B) = \frac{P(0{,}25 < X < 0{,}75)}{P(X > 0{,}25)} = \frac{F(0{,}75) - F(0{,}25)}{1 - F(0{,}25)}$$

Résultat : $$P(X < 0{,}75 \mid X > 0{,}25) = \frac{(0{,}75)^{4/3} - (0{,}25)^{4/3}}{(0{,}75)^{4/3}} = 1 - \frac{(0{,}25)^{4/3}}{(0{,}75)^{4/3}} = 1 - \left(\frac{1}{3}\right)^{4/3}$$ $$\approx 1 - 0{,}242 \approx \boxed{0{,}758}$$
3
Calculer $E(X)$ et $V(X)$
Rappelez les formules de l'espérance et de la variance d'une VA continue, puis calculez-les avec $f(x) = \frac{4}{3}(1-x)^{1/3}$.

Astuce : posez $u = 1-x$ (donc $x = 1-u$) pour obtenir une intégrale en $u$.

Variance : $V(X) = E(X^2) - [E(X)]^2$

Avec $E(X^2) = \displaystyle\int_0^1 x^2 \cdot f(x)\,dx$.

Correction détaillée
Calcul de $E(X)$
$$E(X) = \frac{4}{3}\int_0^1 x(1-x)^{1/3}\,dx$$ Posons $u = 1-x$, $x = 1-u$, $dx = -du$ : $$= \frac{4}{3}\int_0^1 (1-u)\,u^{1/3}\,du = \frac{4}{3}\int_0^1 \left(u^{1/3} - u^{4/3}\right)du$$ $$= \frac{4}{3}\left[\frac{u^{4/3}}{4/3} - \frac{u^{7/3}}{7/3}\right]_0^1 = \frac{4}{3}\left(\frac{3}{4} - \frac{3}{7}\right) = \frac{4}{3} \cdot \frac{21-12}{28}$$ $$= \frac{4}{3} \cdot \frac{9}{28} = \frac{36}{84} = \boxed{\frac{3}{7} \approx 0{,}429}$$
Calcul de $E(X^2)$
$$E(X^2) = \frac{4}{3}\int_0^1 x^2(1-x)^{1/3}\,dx$$ Avec $u = 1-x$ : $$= \frac{4}{3}\int_0^1 (1-u)^2 u^{1/3}\,du = \frac{4}{3}\int_0^1 \left(u^{1/3} - 2u^{4/3} + u^{7/3}\right)du$$ $$= \frac{4}{3}\left[\frac{3}{4} - 2\cdot\frac{3}{7} + \frac{3}{10}\right] = \frac{4}{3}\left[\frac{105 - 120 + 42}{140}\right] = \frac{4}{3}\cdot\frac{27}{140} = \frac{108}{420} = \frac{9}{35}$$
Calcul de $V(X)$
$$V(X) = E(X^2) - [E(X)]^2 = \frac{9}{35} - \left(\frac{3}{7}\right)^2 = \frac{9}{35} - \frac{9}{49}$$ $$= 9\left(\frac{1}{35} - \frac{1}{49}\right) = 9 \cdot \frac{49-35}{35 \times 49} = 9 \cdot \frac{14}{1715} = \frac{126}{1715} = \boxed{\frac{18}{245} \approx 0{,}073}$$

Corrigé complet — exercice 6

1) Détermination de $k$

Conditions d'une densité : $f(x) \geq 0$ et $\displaystyle\int_{-\infty}^{+\infty} f(x)\,dx = 1$.

Positivité : Pour $x \in [0,1]$, $1-x \in [0,1]$, donc $(1-x)^{1/3} \geq 0$ ; avec $k > 0$, $f(x) \geq 0$. ✓

Normalisation : $$\int_0^1 k(1-x)^{1/3}\,dx \overset{u=1-x}{=} k\int_0^1 u^{1/3}\,du = k\cdot\frac{3}{4} = 1 \implies \boxed{k = \frac{4}{3}}$$

2) Fonction de répartition $F(x)$

$$F(x) = \begin{cases} 0 & \text{si } x < 0 \\ 1 - (1-x)^{4/3} & \text{si } 0 \leq x \leq 1 \\ 1 & \text{si } x > 1 \end{cases}$$
a) $P(X \leq 1) = F(1) = 1 - 0 = \boxed{1}$
b) $P(X > \frac{1}{4}) = 1 - F(\frac{1}{4}) = (\frac{3}{4})^{4/3} \approx \boxed{0{,}685}$
c) $P(\frac{1}{4} < X < \frac{1}{2}) = F(\frac{1}{2}) - F(\frac{1}{4}) = (\frac{3}{4})^{4/3} - (\frac{1}{2})^{4/3} \approx \boxed{0{,}288}$
d) $P(X > 3) = \boxed{0}$ car le support de $X$ est $[0,1]$.
e) $P(X < 0{,}75 \mid X > 0{,}25) = \dfrac{(0{,}75)^{4/3} - (0{,}25)^{4/3}}{(0{,}75)^{4/3}} = 1 - (1/3)^{4/3} \approx \boxed{0{,}758}$

3) Espérance et Variance

$$E(X) = \frac{4}{3}\int_0^1 x(1-x)^{1/3}\,dx = \boxed{\frac{3}{7} \approx 0{,}429}$$
$$E(X^2) = \frac{4}{3}\int_0^1 x^2(1-x)^{1/3}\,dx = \frac{9}{35}$$
$$V(X) = E(X^2) - [E(X)]^2 = \frac{9}{35} - \frac{9}{49} = \boxed{\frac{18}{245} \approx 0{,}073}$$
