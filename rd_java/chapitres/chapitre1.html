Module 1

Introduction & Syntaxe de base

Découvrez l'écosystème Java : JDK, JVM, compilation, structure d'un programme et vos premières lignes de code.

Qu'est-ce que Java ?

Imaginez que Java est une langue universelle : vous écrivez une lettre (votre code), un traducteur (le compilateur) la convertit dans un langage intermédiaire (bytecode), puis un interprète local (la JVM) la lit dans chaque pays (système d'exploitation).

Java est un langage de programmation orienté objet, créé en 1995 par Sun Microsystems. Son slogan historique : "Write Once, Run Anywhere". Grâce à la JVM, un programme Java fonctionne sur Windows, Linux, macOS sans modification.

Structure d'un programme Java

Tout programme Java est organisé autour de classes. Voici la structure minimale :

public class Main {
    public static void main(String[] args) {
        // Votre code ici
        System.out.println("Bonjour Java !");
    }
}

Règles essentielles

  • Le fichier doit s'appeler Main.java (même nom que la classe publique)
  • Chaque instruction se termine par un point-virgule ;
  • Les blocs de code sont délimités par des accolades { }
  • Java est sensible à la casse : Main ≠ main
⚠️ Attention : Le nom du fichier .java doit correspondre exactement au nom de la classe publique qu'il contient.

Compilation & Exécution

Java suit un processus en deux étapes :

// Étape 1 : Compilation (code source → bytecode)
javac Main.java     // Produit Main.class

// Étape 2 : Exécution (la JVM lit le bytecode)
java Main            // Affiche : Bonjour Java !
💡 Flux complet : Main.java → javac → Main.class (bytecode) → java (JVM) → résultat affiché.
