Collections : ArrayList & HashMap

Maîtrisez les structures de données dynamiques incontournables en Java.

Pourquoi les Collections ?

Les tableaux classiques sont comme des rangées de casiers fixés au mur : une fois installés, impossible d'en ajouter. Les Collections sont comme des étagères modulables : on ajoute ou retire des éléments à volonté.

Tableau int[]ArrayListHashMap
TailleFixeDynamiqueDynamique
Types primitifsOuiNon (wrappers)Non (wrappers)
Accès parIndexIndexClé
DoublonsOuiOuiClés uniques

Les classes Wrapper

Les Collections ne peuvent pas stocker de types primitifs. On utilise les wrappers :

PrimitifWrapper

💡 Autoboxing : Java convertit automatiquement int ↔ Integer.

ArrayList en détail

import java.util.ArrayList;

ArrayList<String> fruits = new ArrayList<>();

// Ajouter
fruits.add("Pomme");
fruits.add("Banane");
fruits.add("Cerise");
fruits.add(1, "Orange");  // Insérer à l'index 1

// Lire
String premier = fruits.get(0);  // "Pomme"
int taille = fruits.size();  // 4

// Modifier
fruits.set(0, "Fraise");  // Remplace "Pomme"

// Supprimer
fruits.remove("Banane");     // Par valeur
fruits.remove(0);              // Par index

// Vérifier
boolean existe = fruits.contains("Cerise");  // true
int pos = fruits.indexOf("Cerise");  // index ou -1

// Parcourir
for (String f : fruits) {
    System.out.println(f);
}
MéthodeDescription
add(e)Ajoute à la fin
add(i, e)Insère à l'index i
get(i)Retourne l'élément à l'index i
set(i, e)Remplace l'élément à l'index i
remove(i)Supprime par index
remove(obj)Supprime par valeur
size()Nombre d'éléments
contains(e)true si l'élément existe
indexOf(e)Index de l'élément (ou -1)
isEmpty()true si la liste est vide
clear()Vide la liste

HashMap : clé → valeur

Un HashMap associe une clé unique à une valeur. C'est comme un dictionnaire : le mot (clé) donne la définition (valeur).

import java.util.HashMap;

HashMap<String, Integer> notes = new HashMap<>();

// Ajouter des paires clé-valeur
notes.put("Alice", 18);
notes.put("Bob", 14);
notes.put("Clara", 16);

// Lire une valeur
int noteAlice = notes.get("Alice");  // 18
int def = notes.getOrDefault("David", 0);  // 0 (absent)

// Modifier
notes.put("Bob", 15);  // Écrase l'ancienne valeur

// Supprimer
notes.remove("Clara");

// Vérifier
notes.containsKey("Alice");    // true
notes.containsValue(18);       // true

// Taille
notes.size();  // 2

Parcourir un HashMap

import java.util.Map;

// Méthode 1 : Parcourir les clés
for (String nom : notes.keySet()) {
    System.out.println(nom + " : " + notes.get(nom));
}

// Méthode 2 : Parcourir les valeurs
for (int note : notes.values()) {
    System.out.println(note);
}

// Méthode 3 : Parcourir clé + valeur (recommandé)
for (Map.Entry<String, Integer> entry : notes.entrySet()) {
    System.out.println(entry.getKey() + " → " + entry.getValue());
}
Méthode HashMapDescription
put(clé, val)Ajoute ou remplace
get(clé)Retourne la valeur
getOrDefault(clé, def)Valeur ou défaut
remove(clé)Supprime la paire
containsKey(clé)Clé existe ?
containsValue(val)Valeur existe ?
keySet()Ensemble des clés
values()Collection des valeurs
entrySet()Paires clé-valeur
size()Nombre de paires

Exemple pratique : Compteur de mots

import java.util.HashMap;
import java.util.Map;

public class CompteurMots {
    public static void main(String[] args) {
        String phrase = "java est super java est cool";
        String[] mots = phrase.split(" ");

        HashMap<String, Integer> compteur = new HashMap<>();

        for (String mot : mots) {
            int nb = compteur.getOrDefault(mot, 0);
            compteur.put(mot, nb + 1);
        }

        for (Map.Entry<String, Integer> e : compteur.entrySet()) {
            System.out.println(e.getKey() + " : " + e.getValue());
        }
    }
}
// java : 2
// est : 2
// super : 1
// cool : 1
