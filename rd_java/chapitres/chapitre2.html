Module 2

Variables, Types & Opérateurs

Apprenez à déclarer des variables, comprendre les types primitifs et manipuler les opérateurs en Java.

Qu'est-ce qu'une variable ?

Une variable est comme une boîte étiquetée : l'étiquette est le nom, le type de la boîte définit ce qu'elle peut contenir (entier, texte…), et vous pouvez y ranger une valeur.

// Syntaxe : type nom = valeur;
int age = 25;
double prix = 19.99;
String nom = "Alice";
boolean actif = true;
char lettre = 'A';

💡 String n'est PAS un type primitif. C'est une classe (objet). C'est pourquoi il commence par une majuscule.

Constantes (final)

Le mot-clé final rend une variable non modifiable après son initialisation :

final double PI = 3.14159;
final int MAX_ELEVES = 30;
// PI = 3.14; → Erreur de compilation !
⚠️ Convention : Les constantes s'écrivent en MAJUSCULES_AVEC_UNDERSCORES.

Opérateurs

Opérateurs arithmétiques

OpérateurDescriptionExempleRésultat

/Division5 / 31 (division entière)
%Modulo (reste)5 % 32

Opérateurs d'incrémentation

int x = 5;
x++;      // x = 6 (post-incrémentation)
x--;      // x = 5 (post-décrémentation)
x += 3;   // x = 8 (raccourci pour x = x + 3)
x *= 2;   // x = 16

Le casting (conversion de types)

// Casting implicite (petit → grand) : automatique
int a = 10;
double b = a;    // b = 10.0

// Casting explicite (grand → petit) : (type)
double c = 9.7;
int d = (int) c;  // d = 9 (troncature, pas d'arrondi)
🚨 Piège : La division entre deux int donne un int : 5 / 2 = 2, pas 2.5 ! Utilisez 5.0 / 2 pour obtenir 2.5.

Concaténation de chaînes

String prenom = "Alice";
int age = 20;

// Concaténation avec +
String msg = "Je m'appelle " + prenom + " et j'ai " + age + " ans.";
// → "Je m'appelle Alice et j'ai 20 ans."
