Module 3

Affichages & Entrées

Maîtrisez System.out.println, printf et la classe Scanner pour interagir avec l'utilisateur.

Les méthodes d'affichage

System.out est comme un haut-parleur : println parle puis fait une pause (retour à la ligne), print parle sans pause, et printf lit un script formaté (avec des trous à remplir).

println — avec retour à la ligne

System.out.println("Bonjour !");        // Bonjour !
System.out.println("Java");              // Java
System.out.println(42);                  // 42
System.out.println("Age : " + age);     // Age : 25

print — sans retour à la ligne

System.out.print("Bon");
System.out.print("jour");
// Affiche : Bonjour (sur la même ligne)

printf — affichage formaté

String nom = "Alice";
int age = 20;
double note = 15.567;

System.out.printf("Nom : %s, Age : %d%n", nom, age);
System.out.printf("Note : %.2f%n", note);  // Note : 15.57
SpécificateurTypeExemple
%dEntier (int, long)printf("%d", 42)
%fDécimal (double, float)printf("%.2f", 3.14)
%sChaîne (String)printf("%s", "Java")
%cCaractère (char)printf("%c", 'A')
%nRetour à la ligneIndépendant du système

Les séquences d'échappement

SéquenceDescriptionExemple
\nRetour à la ligne"Ligne 1\nLigne 2"
\tTabulation"Col1\tCol2"
\\Antislash"C:\\Users"
\"Guillemet"Il dit \"ok\""

La classe Scanner — lire les entrées

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("Votre nom : ");
        String nom = sc.nextLine();

        System.out.print("Votre âge : ");
        int age = sc.nextInt();

        System.out.println("Bonjour " + nom + ", vous avez " + age + " ans.");

        sc.close();
    }
}

Méthodes de Scanner

MéthodeType luNote
nextLine()String (ligne entière)Lit jusqu'à Entrée
next()String (un mot)S'arrête à l'espace
nextInt()intLève une exception si non entier
nextDouble()doubleAttention virgule/point selon locale
nextBoolean()booleanAttend true ou false
⚠️ Piège classique : Après un nextInt(), le \n reste dans le buffer. Ajoutez un sc.nextLine() avant le prochain nextLine(). 
// Piège et solution
int n = sc.nextInt();
sc.nextLine();  // Consomme le \n restant
String texte = sc.nextLine();  // Fonctionne maintenant
