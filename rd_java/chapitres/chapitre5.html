Module 5

Les Boucles

for, while, do-while et for-each : répétez des instructions efficacement.

La boucle for

La boucle for est comme un compteur automatique : vous définissez le départ, la limite et le pas, puis elle tourne toute seule.

// Syntaxe : for (initialisation; condition; incrémentation)

for (int i = 0; i < 5; i++) {
    System.out.println("i = " + i);
}
// Affiche : i = 0, i = 1, i = 2, i = 3, i = 4

// Compte à rebours
for (int i = 10; i > 0; i--) {
    System.out.println(i);
}

// Pas de 2
for (int i = 0; i <= 10; i += 2) {
    System.out.print(i + " ");  // 0 2 4 6 8 10
}

La boucle while

S'exécute tant que la condition est vraie. La condition est testée avant chaque itération.

int compteur = 0;

while (compteur < 5) {
    System.out.println("Compteur : " + compteur);
    compteur++;
}

// Exemple : chercher un nombre
int n = 1;
while (n * n < 100) {
    n++;
}
System.out.println("Résultat : " + n);  // 10
⚠️ Attention : N'oubliez pas de modifier la variable de condition dans la boucle, sinon vous créez une boucle infinie !

La boucle do-while

Garantit au moins une exécution car la condition est testée après.

Scanner sc = new Scanner(System.in);
int choix;

do {
    System.out.println("1. Jouer");
    System.out.println("2. Quitter");
    System.out.print("Choix : ");
    choix = sc.nextInt();
} while (choix != 2);
💡 Cas d'usage type : Les menus interactifs, la validation de saisie utilisateur (redemander tant que l'entrée est invalide).

La boucle for-each

Parcourt automatiquement chaque élément d'un tableau ou d'une collection :

int[] notes = {12, 15, 18, 9, 14};

for (int note : notes) {
    System.out.println("Note : " + note);
}

// Avec des String
String[] fruits = {"Pomme", "Banane", "Cerise"};
for (String fruit : fruits) {
    System.out.println(fruit);
}
💡 Limitation : Le for-each ne donne pas accès à l'index. Si vous avez besoin de l'index, utilisez un for classique.

break & continue

// break : sort immédiatement de la boucle
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
    System.out.print(i + " ");  // 0 1 2 3 4
}

// continue : passe à l'itération suivante
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) continue;
    System.out.print(i + " ");  // 1 3 5 7 9
}
