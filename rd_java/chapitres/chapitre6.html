Module 6

Tableaux & ArrayList

Stockez et manipulez des collections de données avec les tableaux classiques et ArrayList.

Les tableaux (arrays)

Un tableau est comme une rangée de casiers numérotés. Chaque casier a un numéro (index) et peut contenir un objet du même type.

Déclaration et initialisation

// Déclaration + taille
int[] notes = new int[5];  // 5 cases, toutes à 0

// Déclaration + valeurs
int[] ages = {20, 25, 30, 18};

// Déclaration + new + valeurs
String[] noms = new String[]{"Alice", "Bob", "Charlie"};

Accès et modification

int[] tab = {10, 20, 30, 40};

System.out.println(tab[0]);   // 10 (premier élément)
System.out.println(tab[3]);   // 40 (dernier élément)
tab[1] = 99;               // Modifier le 2e élément
System.out.println(tab.length); // 4 (taille)
🚨 ArrayIndexOutOfBoundsException : Accéder à tab[4] sur un tableau de taille 4 provoque une erreur ! Les index vont de 0 à length-1.

Parcourir un tableau

int[] tab = {5, 10, 15, 20};

// Boucle for classique
for (int i = 0; i < tab.length; i++) {
    System.out.println("Index " + i + " : " + tab[i]);
}

// Boucle for-each (plus simple)
for (int val : tab) {
    System.out.println(val);
}

Tableaux 2D

int[][] matrice = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

System.out.println(matrice[1][2]);  // 6 (ligne 1, colonne 2)

// Parcours avec double boucle
for (int i = 0; i < matrice.length; i++) {
    for (int j = 0; j < matrice[i].length; j++) {
        System.out.print(matrice[i][j] + " ");
    }
    System.out.println();
}

La classe Arrays

import java.util.Arrays;

int[] tab = {5, 2, 8, 1, 9};

// Trier le tableau
Arrays.sort(tab);  // {1, 2, 5, 8, 9}

// Afficher le tableau
System.out.println(Arrays.toString(tab));  // [1, 2, 5, 8, 9]

// Remplir avec une valeur
Arrays.fill(tab, 0);  // {0, 0, 0, 0, 0}

// Copier un tableau
int[] copie = Arrays.copyOf(tab, tab.length);

ArrayList — tableau dynamique

import java.util.ArrayList;

ArrayList<String> fruits = new ArrayList<>();

// Ajouter
fruits.add("Pomme");
fruits.add("Banane");
fruits.add("Cerise");

// Accéder
System.out.println(fruits.get(0));  // Pomme

// Modifier
fruits.set(1, "Mangue");

// Taille
System.out.println(fruits.size());  // 3

// Supprimer
fruits.remove(0);             // Par index
fruits.remove("Cerise");      // Par valeur

// Vérifier la présence
boolean existe = fruits.contains("Mangue");  // true

// Parcourir
for (String f : fruits) {
    System.out.println(f);
}
