Module 8

Classes & Objets

Introduction à la Programmation Orientée Objet : classes, objets, constructeurs et le mot-clé this.

Classe vs Objet

Une classe est un plan d'architecte : elle décrit les pièces et les dimensions. Un objet est la maison construite à partir de ce plan. On peut construire plusieurs maisons (objets) à partir du même plan (classe).

ClasseObjet
Plan / modèleInstance concrète
Définit les attributs et méthodesPossède ses propres valeurs
N'existe qu'une foisPeut exister en plusieurs exemplaires
class Voiture { }Voiture v = new Voiture();

Créer une classe

public class Voiture {
    // Attributs (propriétés)
    String marque;
    String couleur;
    int vitesse;

    // Méthode
    void accelerer(int ajout) {
        vitesse += ajout;
        System.out.println(marque + " roule à " + vitesse + " km/h");
    }

    void freiner() {
        vitesse = 0;
        System.out.println(marque + " s'est arrêtée.");
    }
}

Créer et utiliser un objet

public class Main {
    public static void main(String[] args) {
        // Création d'un objet
        Voiture v1 = new Voiture();
        v1.marque = "Toyota";
        v1.couleur = "Rouge";

        // Utilisation
        v1.accelerer(50);   // Toyota roule à 50 km/h
        v1.accelerer(30);   // Toyota roule à 80 km/h
        v1.freiner();        // Toyota s'est arrêtée.

        // Deuxième objet indépendant
        Voiture v2 = new Voiture();
        v2.marque = "BMW";
    }
}

Les constructeurs

Un constructeur est une méthode spéciale qui initialise l'objet lors de sa création avec new.

public class Voiture {
    String marque;
    String couleur;
    int vitesse;

    // Constructeur avec paramètres
    public Voiture(String marque, String couleur) {
        this.marque = marque;     // this.marque = attribut
        this.couleur = couleur;   // couleur = paramètre
        this.vitesse = 0;
    }

    // Constructeur par défaut (sans paramètres)
    public Voiture() {
        this.marque = "Inconnue";
        this.couleur = "Grise";
        this.vitesse = 0;
    }
}

// Utilisation
Voiture v1 = new Voiture("Tesla", "Blanc");
Voiture v2 = new Voiture();  // Constructeur par défaut

Règles des constructeurs

  • Le nom du constructeur = le nom de la classe
  • Pas de type de retour (même pas void)
  • Appelé automatiquement avec new
  • On peut avoir plusieurs constructeurs (surcharge)

Le mot-clé this

this fait référence à l'objet courant. Il permet de distinguer un attribut d'un paramètre portant le même nom.

public Voiture(String marque) {
    this.marque = marque;
    // this.marque → l'attribut de l'objet
    // marque     → le paramètre reçu
}
💡 Bonne pratique : Utilisez this dans les constructeurs et les setters pour lever l'ambiguïté entre attribut et paramètre.
