TP2 — Calculatrice de base

Contexte : Demander 2 nombres à l'utilisateur et afficher les 4 opérations.

Objectifs

  • Utiliser Scanner pour lire 2 doubles
  • Afficher la somme, différence, produit, quotient
  • Gérer la division par zéro
📝 Voir la correction 
import java.util.Scanner;

public class Calculatrice {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("Nombre 1 : ");
        double a = sc.nextDouble();

        System.out.print("Nombre 2 : ");
        double b = sc.nextDouble();

        System.out.printf("%.2f + %.2f = %.2f%n", a, b, a + b);
        System.out.printf("%.2f - %.2f = %.2f%n", a, b, a - b);
        System.out.printf("%.2f × %.2f = %.2f%n", a, b, a * b);

        if (b != 0) {
            System.out.printf("%.2f ÷ %.2f = %.2f%n", a, b, a / b);
        } else {
            System.out.println("Division par zéro impossible !");
        }
    }
}

TP4 — Jeu de devinette

Contexte : L'ordinateur choisit un nombre entre 1 et 100. L'utilisateur doit le deviner avec des indices.

📝 Voir la correction 
import java.util.Scanner;
import java.util.Random;

public class Devinette {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int secret = new Random().nextInt(100) + 1;
        int essais = 0;
        int guess;

        System.out.println("🎯 Devinez le nombre (1-100) !");

        do {
            System.out.print("Votre essai : ");
            guess = sc.nextInt();
            essais++;

            if (guess < secret) {
                System.out.println("⬆️ Plus grand !");
            } else if (guess > secret) {
                System.out.println("⬇️ Plus petit !");
            }
        } while (guess != secret);

        System.out.println("🎉 Bravo ! Trouvé en " + essais + " essais.");
    }
}

TP5 — Dessiner des motifs avec des boucles

Contexte : Utilisez des boucles imbriquées pour dessiner un triangle d'étoiles.

📝 Voir la correction 
public class Triangle {
    public static void main(String[] args) {
        int n = 5;

        // Triangle aligné à gauche
        for (int i = 1; i <= n; i++) {
            for (int j = 0; j < i; j++) {
                System.out.print("* ");
            }
            System.out.println();
        }

        System.out.println();

        // Pyramide centrée
        for (int i = 1; i <= n; i++) {
            for (int j = 0; j < n - i; j++) System.out.print(" ");
            for (int j = 0; j < 2 * i - 1; j++) System.out.print("*");
            System.out.println();
        }
    }
}

TP6 — Gestion de notes (tableau)

Contexte : Stocker les notes d'une classe, calculer la moyenne, le min, le max et le nombre d'admis.

📝 Voir la correction 
public class GestionNotes {
    public static void main(String[] args) {
        double[] notes = {15.5, 8.0, 12.0, 17.5, 9.5, 14.0, 6.0, 18.0, 11.5, 13.0};

        double somme = 0, min = notes[0], max = notes[0];
        int admis = 0;

        for (double n : notes) {
            somme += n;
            if (n < min) min = n;
            if (n > max) max = n;
            if (n >= 10) admis++;
        }

        System.out.printf("Moyenne : %.2f%n", somme / notes.length);
        System.out.println("Min : " + min + " | Max : " + max);
        System.out.println("Admis : " + admis + "/" + notes.length);
    }
}

TP7 — Bibliothèque de fonctions mathématiques

Contexte : Créer des fonctions utilitaires : factorielle, puissance, estPremier, PGCD.

📝 Voir la correction 
public class MathUtils {

    public static long factorielle(int n) {
        long result = 1;
        for (int i = 2; i <= n; i++) result *= i;
        return result;
    }

    public static double puissance(double base, int exp) {
        double result = 1;
        for (int i = 0; i < Math.abs(exp); i++) result *= base;
        return exp >= 0 ? result : 1.0 / result;
    }

    public static boolean estPremier(int n) {
        if (n < 2) return false;
        for (int i = 2; i <= Math.sqrt(n); i++) {
            if (n % i == 0) return false;
        }
        return true;
    }

    public static int pgcd(int a, int b) {
        while (b != 0) {
            int temp = b;
            b = a % b;
            a = temp;
        }
        return a;
    }

    public static void main(String[] args) {
        System.out.println("5! = " + factorielle(5));    // 120
        System.out.println("2^10 = " + puissance(2, 10)); // 1024
        System.out.println("17 premier ? " + estPremier(17)); // true
        System.out.println("PGCD(48,18) = " + pgcd(48, 18)); // 6
    }
}

TP8 — Classe Produit (POO basique)

Contexte : Créer une classe Produit avec nom, prix et quantité. Gérer un panier d'achats.

📝 Voir la correction 
public class Produit {
    String nom;
    double prix;
    int quantite;

    public Produit(String nom, double prix, int quantite) {
        this.nom = nom;
        this.prix = prix;
        this.quantite = quantite;
    }

    double total() {
        return prix * quantite;
    }

    void afficher() {
        System.out.printf("%-15s %6.2f € x%d = %8.2f €%n",
            nom, prix, quantite, total());
    }

    public static void main(String[] args) {
        Produit[] panier = {
            new Produit("Clavier", 49.99, 1),
            new Produit("Souris", 25.50, 2),
            new Produit("Écran", 299.00, 1)
        };

        double grandTotal = 0;
        for (Produit p : panier) {
            p.afficher();
            grandTotal += p.total();
        }
        System.out.printf("%nTOTAL : %.2f €%n", grandTotal);
    }
}

TP9 — Compte bancaire encapsulé

Contexte : Créer une classe CompteBancaire avec encapsulation complète, validation et compteur static.

📝 Voir la correction 
public class CompteBancaire {
    private String titulaire;
    private double solde;
    private static int nbComptes = 0;
    private int numero;

    public CompteBancaire(String titulaire, double soldeInitial) {
        this.titulaire = titulaire;
        this.solde = Math.max(0, soldeInitial);
        nbComptes++;
        this.numero = nbComptes;
    }

    public void deposer(double montant) {
        if (montant > 0) {
            solde += montant;
            System.out.printf("✅ +%.2f € → Solde: %.2f €%n", montant, solde);
        }
    }

    public void retirer(double montant) {
        if (montant > 0 && montant <= solde) {
            solde -= montant;
            System.out.printf("✅ -%.2f € → Solde: %.2f €%n", montant, solde);
        } else {
            System.out.println("❌ Solde insuffisant !");
        }
    }

    public double getSolde() { return solde; }
    public static int getNbComptes() { return nbComptes; }

    @Override
    public String toString() {
        return "Compte #" + numero + " (" + titulaire + ") : " + solde + " €";
    }

    public static void main(String[] args) {
        CompteBancaire c1 = new CompteBancaire("Alice", 1000);
        CompteBancaire c2 = new CompteBancaire("Bob", 500);

        c1.deposer(200);
        c1.retirer(50);
        c2.retirer(600);  // Solde insuffisant

        System.out.println(c1);
        System.out.println(c2);
        System.out.println("Total comptes : " + CompteBancaire.getNbComptes());
    }
}

TP10 — Annuaire avec HashMap

Contexte : Créer un annuaire téléphonique avec HashMap. Menu interactif : ajouter, chercher, lister, supprimer.

📝 Voir la correction 
import java.util.HashMap;
import java.util.Map;
import java.util.Scanner;

public class Annuaire {
    public static void main(String[] args) {
        HashMap<String, String> contacts = new HashMap<>();
        Scanner sc = new Scanner(System.in);
        int choix;

        do {
            System.out.println("\n=== ANNUAIRE ===");
            System.out.println("1. Ajouter");
            System.out.println("2. Chercher");
            System.out.println("3. Lister tous");
            System.out.println("4. Supprimer");
            System.out.println("0. Quitter");
            System.out.print("Choix : ");
            choix = sc.nextInt(); sc.nextLine();

            switch (choix) {
                case 1:
                    System.out.print("Nom : ");
                    String nom = sc.nextLine();
                    System.out.print("Téléphone : ");
                    String tel = sc.nextLine();
                    contacts.put(nom, tel);
                    System.out.println("✅ Ajouté !");
                    break;
                case 2:
                    System.out.print("Nom à chercher : ");
                    String search = sc.nextLine();
                    String result = contacts.getOrDefault(search, "Non trouvé");
                    System.out.println(search + " → " + result);
                    break;
                case 3:
                    for (Map.Entry<String,String> e : contacts.entrySet()) {
                        System.out.println(e.getKey() + " : " + e.getValue());
                    }
                    break;
                case 4:
                    System.out.print("Nom à supprimer : ");
                    contacts.remove(sc.nextLine());
                    break;
            }
        } while (choix != 0);
    }
}

TP11 — Statistiques ArrayList

Contexte : L'utilisateur saisit des nombres. Calculer moyenne, écart-type, trier et afficher l'histogramme.

📝 Voir la correction 
import java.util.ArrayList;
import java.util.Collections;
import java.util.Scanner;

public class Statistiques {
    public static void main(String[] args) {
        ArrayList<Double> nombres = new ArrayList<>();
        Scanner sc = new Scanner(System.in);

        System.out.println("Entrez des nombres (0 pour terminer) :");
        while (true) {
            double n = sc.nextDouble();
            if (n == 0) break;
            nombres.add(n);
        }

        if (nombres.isEmpty()) { System.out.println("Aucune donnée."); return; }

        // Moyenne
        double somme = 0;
        for (double v : nombres) somme += v;
        double moyenne = somme / nombres.size();

        // Écart-type
        double variance = 0;
        for (double v : nombres) variance += (v - moyenne) * (v - moyenne);
        double ecartType = Math.sqrt(variance / nombres.size());

        Collections.sort(nombres);

        System.out.printf("Moyenne : %.2f%n", moyenne);
        System.out.printf("Écart-type : %.2f%n", ecartType);
        System.out.println("Min : " + nombres.get(0));
        System.out.println("Max : " + nombres.get(nombres.size() - 1));
        System.out.println("Trié : " + nombres);

        // Histogramme texte
        double max = nombres.get(nombres.size() - 1);
        System.out.println("\nHistogramme :");
        for (double v : nombres) {
            int barLen = (int) (v / max * 30);
            System.out.printf("%6.1f | %s%n", v, "█".repeat(barLen));
        }
    }
}

TP12 — Mini-projet : Gestionnaire d'étudiants

Contexte : Projet final combinant classes, encapsulation, ArrayList et HashMap. Gérer des étudiants et leurs notes.

📝 Voir la correction 
import java.util.*;

public class Etudiant {
    private String nom;
    private ArrayList<Double> notes;
    private static int compteur = 0;
    private int id;

    public Etudiant(String nom) {
        this.nom = nom;
        this.notes = new ArrayList<>();
        this.id = ++compteur;
    }

    public void ajouterNote(double note) {
        if (note >= 0 && note <= 20) notes.add(note);
    }

    public double getMoyenne() {
        if (notes.isEmpty()) return 0;
        double s = 0;
        for (double n : notes) s += n;
        return s / notes.size();
    }

    public boolean estAdmis() { return getMoyenne() >= 10; }
    public String getNom() { return nom; }

    @Override
    public String toString() {
        return String.format("#%d %s | Notes: %s | Moy: %.2f | %s",
            id, nom, notes, getMoyenne(),
            estAdmis() ? "✅ Admis" : "❌ Recalé");
    }

    public static void main(String[] args) {
        ArrayList<Etudiant> promo = new ArrayList<>();

        Etudiant e1 = new Etudiant("Alice");
        e1.ajouterNote(15); e1.ajouterNote(17); e1.ajouterNote(14);

        Etudiant e2 = new Etudiant("Bob");
        e2.ajouterNote(8); e2.ajouterNote(9); e2.ajouterNote(7);

        Etudiant e3 = new Etudiant("Clara");
        e3.ajouterNote(12); e3.ajouterNote(10); e3.ajouterNote(11);

        promo.add(e1); promo.add(e2); promo.add(e3);

        System.out.println("=== BULLETIN DE LA PROMO ===");
        for (Etudiant e : promo) System.out.println(e);

        // Statistiques avec HashMap
        HashMap<String, Double> moyennes = new HashMap<>();
        for (Etudiant e : promo) {
            moyennes.put(e.getNom(), e.getMoyenne());
        }

        System.out.println("\n=== MOYENNES ===");
        for (Map.Entry<String,Double> e : moyennes.entrySet()) {
            System.out.printf("%s : %.2f%n", e.getKey(), e.getValue());
        }
    }
}
