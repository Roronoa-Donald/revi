Chapitre 2

Vite et JSX

Lire la structure du projet et ecrire du JSX propre.

Structure Vite

index.html contient la balise root. src/main.jsx monte React dans ce root. App.jsx est le premier composant visible.

Au debut, ne modifie main.jsx que si tu sais pourquoi. Travaille surtout dans App.jsx et dans de nouveaux composants.

FichierRole
index.htmlPoint d entree HTML avec div id root
src/main.jsxDemarre React avec createRoot
src/App.jsxComposant principal
src/App.cssStyles du composant principal
Memo rapide : index.html accueille, main.jsx branche, App.jsx affiche.

JSX

JSX ressemble a HTML mais reste du JavaScript. Les accolades permettent d inserer une expression.

La regle de base: retourner un seul element parent, ou utiliser un fragment vide.

  • className remplace class.
  • Les balises sans enfant se ferment: <img />.
  • Le style inline est un objet: style={{ color: "red" }}.
function Bonjour() {
  const nom = "Awa";
  return <h1>Bonjour, {nom}</h1>;
}
Piege : Dans JSX, tu ecris className. Si tu mets class, tu melanges HTML brut et JSX.

Fragment

Un composant ne peut pas retourner deux balises voisines sans parent. Le fragment regroupe sans ajouter de div inutile.

C est pratique pour eviter une structure HTML lourde.

return (
  <>
    <h1>Titre</h1>
    <p>Texte</p>
  </>
);
Memo rapide : Fragment = boite invisible.
