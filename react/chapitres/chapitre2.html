Chapitre 2

Vite et JSX

Lire la structure du projet et ecrire du JSX propre.

Structure Vite

index.html contient la balise root. src/main.jsx monte React dans ce root. App.jsx est le premier composant visible.

Au debut, ne modifie main.jsx que si tu sais pourquoi. Travaille surtout dans App.jsx et dans de nouveaux composants.

FichierRole
index.htmlPoint d entree HTML avec div id root
src/main.jsxDemarre React avec createRoot
src/App.jsxComposant principal
src/App.cssStyles du composant principal
Memo rapide : index.html accueille, main.jsx branche, App.jsx affiche.

JSX

JSX ressemble a HTML mais reste du JavaScript. Les accolades permettent d inserer une expression.

La regle de base: retourner un seul element parent, ou utiliser un fragment vide.

  • className remplace class.
  • Les balises sans enfant se ferment: <img />.
  • Le style inline est un objet: style={{ color: "red" }}.
function Bonjour() {
  const nom = "Awa";
  return <h1>Bonjour, {nom}</h1>;
}
Piege : Dans JSX, tu ecris className. Si tu mets class, tu melanges HTML brut et JSX.

Fragment

Un composant ne peut pas retourner deux balises voisines sans parent. Le fragment regroupe sans ajouter de div inutile.

C est pratique pour eviter une structure HTML lourde.

return (
  <>
    <h1>Titre</h1>
    <p>Texte</p>
  </>
);
Memo rapide : Fragment = boite invisible.

Questions de fin de chapitre

Reponds avant de lire la correction. Elle se debloque apres une bonne reponse, ou apres deux erreurs.

1. Dans un projet Vite, quel fichier contient la balise '<div id="root"></div>' qui accueille l'application ?
Correction : Vite place l'index.html à la racine du projet, qui sert de point d'entrée pour charger le script main.jsx.
2. À quoi sert le fichier 'src/main.jsx' dans la structure standard de Vite ?
Correction : main.jsx initialise le point de greffe (root) de l'arbre React à l'aide de react-dom/client.
3. Qu'est-ce que JSX ?
Correction : JSX (JavaScript XML) permet de structurer l'affichage de manière intuitive tout en conservant toute la puissance de calcul du JS.
4. Dans JSX, comment insère-t-on une expression ou une variable JavaScript ?
Correction : Les accolades indiquent au compilateur JSX d'évaluer le contenu comme du code JavaScript standard.
5. Quelle règle de structure majeure doit respecter le retour JSX d'un composant ?
Correction : Une fonction JavaScript ne peut retourner qu'une seule valeur. En JSX, cela se traduit par un unique élément parent englobant.
