Chapitre 3

Composants et props

Passer des donnees sans casser la reutilisabilite.

Composant fonctionnel

Un composant React commence par une majuscule. Techniquement, c est une fonction JavaScript qui retourne du JSX.

Tu peux l appeler plusieurs fois avec des donnees differentes.

function Carte({ titre, description }) {
  return (
    <article className="carte">
      <h2>{titre}</h2>
      <p>{description}</p>
    </article>
  );
}
Tip : Majuscule = React comprend que c est un composant, pas une balise HTML.

Props

Les props sont des parametres envoyes par le parent. Elles sont immutables: l enfant les lit, il ne les modifie pas.

La destructuration rend le code plus lisible quand tu sais quelles props tu attends.

Memo rapide : Props = colis envoye du parent vers l enfant.

children

props.children represente le contenu place entre les balises d un composant.

C est utile pour creer un composant Conteneur, Alerte ou Layout.

function Conteneur({ children }) {
  return <div className="conteneur">{children}</div>;
}

<Conteneur><p>Texte interne</p></Conteneur>
Piege : Ne mets pas tout dans un seul composant App. Des composants courts donnent un code plus lisible et plus testable.

Questions de fin de chapitre

Reponds avant de lire la correction. Elle se debloque apres une bonne reponse, ou apres deux erreurs.

1. Quelle règle de casse doit respecter le nom d'un composant personnalisé ?
Correction : React utilise la première lettre : une minuscule indique une balise HTML standard (ex: <div>), une majuscule cible un composant.
2. Que sont les 'props' en React ?
Correction : Les props (propriétés) permettent de personnaliser l'affichage et le comportement d'un composant enfant depuis son parent.
3. Comment caractérise-t-on la mutabilité des 'props' au sein du composant enfant qui les reçoit ?
Correction : Le flux de données de React est unidirectionnel. Un enfant ne doit pas modifier ses props ; il doit notifier le parent s'il veut un changement.
4. Quelle syntaxe JavaScript moderne permet de lire proprement les props à l'entrée d'un composant ?
Correction : La destructuration permet d'extraire directement les clés de l'objet props sous forme de variables utilisables dans le composant.
5. Que représente la propriété spéciale 'props.children' ?
Correction : `children` permet de créer des composants d'affichage enveloppants (layouts, cartes, modales) qui englobent du contenu libre.
