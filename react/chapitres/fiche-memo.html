Fiche memo

Les idees a ressortir vite en controle, sans perdre le raisonnement.

Formules mentales

  • Composant = fonction + JSX.
  • Props = donnees qui descendent.
  • State = donnees qui changent.
  • Rendu = React reconstruit l interface depuis les donnees.

Syntaxes a retenir

  • useState: const [x, setX] = useState(valeurInitiale).
  • Liste: items.map(item => <li key={item.id}>...</li>).
  • Effet: useEffect(() => { ... }, [dependances]).
  • Route 404: <Route path="*" element={<NotFound />} />.

Pieges classiques

  • Ne pas modifier le state directement.
  • Ne pas utiliser index comme key si la liste change.
  • Ne pas oublier value/onChange dans un formulaire controle.
  • Ne pas creer une boucle infinie avec useEffect.
