Roadmap PHP vs C# — Classes & Objets TP PHP

Classes & Objets

Le cœur de la POO : déclarer une classe, créer un objet, écrire un constructeur — exercice 1 de l'examen.

Déclarer une classe

La syntaxe est presque identique. La grande différence : PHP utilise $ pour les propriétés et -> pour y accéder.

PHP
class Livre {
    public $titre;
    public $auteur;
    public $prix;
}

// Créer un objet
$livre = new Livre();
$livre->titre = "PHP POO";
$livre->auteur = "Alice";
$livre->prix = 25.50;

echo $livre->titre; // PHP POO
C#
class Livre {
    public string Titre;
    public string Auteur;
    public double Prix;
}

// Créer un objet
Livre livre = new Livre();
livre.Titre = "C# POO";
livre.Auteur = "Alice";
livre.Prix = 25.50;

Console.WriteLine(livre.Titre); // C# POO
⚡ Mémo visuel :
• PHP : $livre->titre (flèche)
• C# : livre.Titre (point)
• PHP : pas de type devant les propriétés (ou optionnel en PHP 8)
• C# : type OBLIGATOIRE (string, int, double…)

Constructeur

Le constructeur initialise l'objet. PHP utilise un nom magique __construct, C# utilise le nom de la classe.

PHP
class Livre {
    public $titre;
    public $auteur;
    public $prix;

    // Constructeur magique
    public function __construct($titre, $auteur, $prix) {
        $this->titre = $titre;
        $this->auteur = $auteur;
        $this->prix = $prix;
    }
}

// Utilisation
$livre = new Livre("PHP POO", "Alice", 25.50);
echo $livre->titre; // PHP POO
C#
class Livre {
    public string Titre;
    public string Auteur;
    public double Prix;

    // Constructeur = nom de la classe
    public Livre(string titre, string auteur, double prix) {
        Titre = titre;    // this. optionnel
        Auteur = auteur;
        Prix = prix;
    }
}

// Utilisation
Livre livre = new Livre("C# POO", "Alice", 25.50);
Console.WriteLine(livre.Titre); // C# POO
💡 Astuce : En PHP, $this-> est OBLIGATOIRE pour accéder aux propriétés. En C#, this. est optionnel sauf si le nom du paramètre est identique à la propriété.

Méthodes

Les méthodes sont les fonctions d'une classe. Même logique, syntaxe proche.

PHP
class Livre {
    public $titre;
    public $prix;
    public $quantite;

    public function __construct($titre, $prix, $qte) {
        $this->titre = $titre;
        $this->prix = $prix;
        $this->quantite = $qte;
    }

    // Méthode de calcul
    public function prixTotal() {
        return $this->prix * $this->quantite;
    }

    // Méthode d'affichage
    public function __toString() {
        return "{$this->titre} — {$this->prix}€ x{$this->quantite}";
    }
}

$l = new Livre("PHP", 25, 3);
echo $l->prixTotal(); // 75
echo $l;  // PHP — 25€ x3
C#
class Livre {
    public string Titre;
    public double Prix;
    public int Quantite;

    public Livre(string titre, double prix, int qte) {
        Titre = titre;
        Prix = prix;
        Quantite = qte;
    }

    // Méthode de calcul
    public double PrixTotal() {
        return Prix * Quantite;
    }

    // Équivalent de __toString()
    public override string ToString() {
        return $"{Titre} — {Prix}€ x{Quantite}";
    }
}

Livre l = new Livre("C#", 25, 3);
Console.WriteLine(l.PrixTotal()); // 75
Console.WriteLine(l); // C# — 25€ x3
⚡ Différence clé — ToString :
• PHP: public function __toString() — méthode magique
• C#: public override string ToString() — override obligatoire car on surcharge la méthode héritée de Object

Tableaux d'objets

Le pattern classique de l'examen : créer plusieurs objets et les stocker dans un tableau.

PHP
$livres = [
    new Livre("PHP POO", 25, 3),
    new Livre("SQL", 30, 1),
    new Livre("JS", 20, 5),
];

// Parcourir
foreach ($livres as $livre) {
    echo $livre->titre . " : " 
         . $livre->prixTotal() . "€\n";
}

// Calculer le total global
$total = 0;
foreach ($livres as $livre) {
    $total += $livre->prixTotal();
}
echo "Total : {$total}€";
C#
List<Livre> livres = new() {
    new Livre("C# POO", 25, 3),
    new Livre("SQL", 30, 1),
    new Livre("JS", 20, 5),
};

// Parcourir
foreach (Livre livre in livres) {
    Console.WriteLine($"{livre.Titre} : " +
        $"{livre.PrixTotal()}€");
}

// Calculer le total global
double total = 0;
foreach (Livre livre in livres) {
    total += livre.PrixTotal();
}
Console.WriteLine($"Total : {total}€");

Exercice type examen — Classe Livre (PHP + C#)

Reproduis l'exercice 1 du partiel dans les deux langages :

Consigne (inspirée examen 2024/2025) :
1. Crée une classe Livre avec : titre (string), auteur (string), prix (float/double)
2. Constructeur qui initialise les 3 propriétés
3. Méthode afficher() : retourne "Titre par Auteur — Prix €"
4. Crée 3 livres dans un tableau et affiche-les avec une boucle
PHP
<?php
class Livre {
    public $titre;
    public $auteur;
    public $prix;

    public function __construct($titre, $auteur, $prix) {
        $this->titre = $titre;
        $this->auteur = $auteur;
        $this->prix = $prix;
    }

    public function afficher() {
        return "{$this->titre} par {$this->auteur} — {$this->prix} €";
    }
}

$livres = [
    new Livre("PHP POO", "Dupont", 29.90),
    new Livre("SQL Avancé", "Martin", 35.00),
    new Livre("UML", "Durand", 24.50)
];

foreach ($livres as $livre) {
    echo $livre->afficher() . "\n";
}
C#
class Livre {
    public string Titre;
    public string Auteur;
    public double Prix;

    public Livre(string titre, string auteur, double prix) {
        Titre = titre;
        Auteur = auteur;
        Prix = prix;
    }

    public string Afficher() {
        return $"{Titre} par {Auteur} — {Prix} €";
    }
}

// Programme principal
List<Livre> livres = new() {
    new Livre("C# POO", "Dupont", 29.90),
    new Livre("SQL Avancé", "Martin", 35.00),
    new Livre("UML", "Durand", 24.50)
};

foreach (Livre livre in livres) {
    Console.WriteLine(livre.Afficher());
}
Pourquoi c'est important : C'est l'exercice 1 du partiel (5 pts). Le pattern est toujours le même : classe + constructeur + méthode d'affichage + tableau d'objets. Si tu sais le faire en PHP, tu sais le faire en C# — seule la syntaxe change (-> vs ., $ vs types, __construct vs nom de classe).
