Héritage simple

La classe enfant récupère TOUT ce qui est public/protected du parent.

PHP
class Livre {
    private $titre;
    private $auteur;
    private $prix;

    public function __construct($titre, $auteur, $prix) {
        $this->titre = $titre;
        $this->auteur = $auteur;
        $this->setPrix($prix);
    }

    public function getTitre() { return $this->titre; }
    public function getAuteur() { return $this->auteur; }
    public function getPrix() { return $this->prix; }
    public function setPrix($p) {
        if ($p < 0) throw new Exception("Prix < 0");
        $this->prix = $p;
    }

    public function afficher() {
        return "{$this->titre} par {$this->auteur} — {$this->prix}€";
    }
}

// HÉRITAGE avec extends
class LivreElectronique extends Livre {
    private $tailleFichier;

    public function __construct($titre, $auteur, $prix, $taille) {
        parent::__construct($titre, $auteur, $prix);
        $this->tailleFichier = $taille;
    }

    public function getTailleFichier() {
        return $this->tailleFichier;
    }

    // SURCHARGE (override)
    public function afficher() {
        return parent::afficher()
             . " [PDF: {$this->tailleFichier} Mo]";
    }
}

$e = new LivreElectronique("PHP", "Alice", 15, 2.5);
echo $e->afficher();
// PHP par Alice — 15€ [PDF: 2.5 Mo]
C#
class Livre {
    public string Titre { get; private set; }
    public string Auteur { get; private set; }
    private double prix;
    public double Prix {
        get => prix;
        set {
            if (value < 0) throw new Exception("Prix < 0");
            prix = value;
        }
    }

    public Livre(string titre, string auteur, double prix) {
        Titre = titre;
        Auteur = auteur;
        Prix = prix;
    }

    // virtual = peut être surchargée
    public virtual string Afficher() {
        return $"{Titre} par {Auteur} — {Prix}€";
    }
}

// HÉRITAGE avec :
class LivreElectronique : Livre {
    public double TailleFichier { get; private set; }

    public LivreElectronique(string titre, string auteur,
        double prix, double taille)
        : base(titre, auteur, prix)   // appel parent
    {
        TailleFichier = taille;
    }

    // override OBLIGATOIRE
    public override string Afficher() {
        return base.Afficher()
             + $" [PDF: {TailleFichier} Mo]";
    }
}

var e = new LivreElectronique("C#", "Alice", 15, 2.5);
Console.WriteLine(e.Afficher());
// C# par Alice — 15€ [PDF: 2.5 Mo]
⚡ Les 3 piège C# à retenir :
1. virtual dans le parent = "je permets la surcharge"
2. override dans l'enfant = "je surcharge"
3. : base() après la signature du constructeur (pas dans le corps)
En PHP, rien de tout ça → on surcharge juste en redéfinissant la méthode.

Classes & Méthodes Abstraites

Une classe abstraite ne peut pas être instanciée. Elle force les enfants à implémenter certaines méthodes.

PHP
abstract class Forme {
    protected $couleur;

    public function __construct($couleur) {
        $this->couleur = $couleur;
    }

    // Méthode abstraite = PAS de corps
    abstract public function aire();

    // Méthode concrète = partagée
    public function decrire() {
        return "Forme {$this->couleur}, "
             . "aire=" . $this->aire();
    }
}

class Cercle extends Forme {
    private $rayon;

    public function __construct($couleur, $rayon) {
        parent::__construct($couleur);
        $this->rayon = $rayon;
    }

    // OBLIGÉ d'implémenter aire()
    public function aire() {
        return round(M_PI * $this->rayon ** 2, 2);
    }
}

$c = new Cercle("rouge", 5);
echo $c->decrire(); // Forme rouge, aire=78.54
// new Forme("bleu"); ❌ ERREUR
C#
abstract class Forme {
    protected string Couleur;

    public Forme(string couleur) {
        Couleur = couleur;
    }

    // Méthode abstraite
    public abstract double Aire();

    // Méthode concrète
    public string Decrire() {
        return $"Forme {Couleur}, aire={Aire()}";
    }
}

class Cercle : Forme {
    private double rayon;

    public Cercle(string couleur, double rayon)
        : base(couleur)
    {
        this.rayon = rayon;
    }

    // override OBLIGATOIRE
    public override double Aire() {
        return Math.Round(Math.PI * rayon * rayon, 2);
    }
}

Cercle c = new Cercle("rouge", 5);
Console.WriteLine(c.Decrire());
// new Forme("bleu"); ❌ ERREUR

Interfaces

Un contrat : la classe DOIT implémenter toutes les méthodes de l'interface.

PHP
interface Affichable {
    public function afficher();
}

interface Calculable {
    public function calculer();
}

// implements (peut en avoir plusieurs)
class Facture implements Affichable, Calculable {
    private $montant;

    public function __construct($montant) {
        $this->montant = $montant;
    }

    public function afficher() {
        return "Facture : {$this->montant}€";
    }

    public function calculer() {
        return $this->montant * 1.20; // TTC
    }
}
C#
interface IAffichable {
    string Afficher();
}

interface ICalculable {
    double Calculer();
}

// : (même syntaxe que l'héritage)
class Facture : IAffichable, ICalculable {
    private double montant;

    public Facture(double montant) {
        this.montant = montant;
    }

    public string Afficher() {
        return $"Facture : {montant}€";
    }

    public double Calculer() {
        return montant * 1.20; // TTC
    }
}
🔑 Convention C# : Les interfaces commencent par I majuscule : IAffichable, ICalculable. En PHP, pas de convention imposée (mais Affichable est courant).
⚡ Piège C# : Pour hériter + implémenter :
• PHP : class B extends A implements I
• C# : class B : A, I (classe parent EN PREMIER, interfaces après)

Exercice examen — LivreElectronique (PHP + C#)

Consigne (examen 2024/2025, exercice 3 — 5 pts) :
1. Crée une classe LivreElectronique qui hérite de Livre
2. Ajoute un attribut tailleFichier (float/double) avec getter
3. Le constructeur appelle le constructeur parent + initialise tailleFichier
4. Surcharge afficher() pour ajouter "[PDF: X Mo]"
5. Crée 2 objets (un Livre, un LivreElectronique) et affiche-les
PHP
<?php
// Livre (classe parent — déjà faite au TP3)
class Livre {
    private $titre;
    private $auteur;
    private $prix;

    public function __construct($t, $a, $p) {
        $this->titre = $t;
        $this->auteur = $a;
        $this->setPrix($p);
    }
    public function getTitre() { return $this->titre; }
    public function getAuteur() { return $this->auteur; }
    public function getPrix() { return $this->prix; }
    public function setPrix($p) {
        if ($p < 0) throw new Exception("Prix < 0");
        $this->prix = $p;
    }
    public function afficher() {
        return "{$this->titre} par {$this->auteur} — {$this->prix}€";
    }
}

// LivreElectronique (enfant)
class LivreElectronique extends Livre {
    private $tailleFichier;

    public function __construct($t, $a, $p, $taille) {
        parent::__construct($t, $a, $p);
        $this->tailleFichier = $taille;
    }

    public function getTailleFichier() {
        return $this->tailleFichier;
    }

    public function afficher() {
        return parent::afficher()
             . " [PDF: {$this->tailleFichier} Mo]";
    }
}

// Test
$l1 = new Livre("PHP POO", "Dupont", 29.90);
$l2 = new LivreElectronique("SQL", "Martin", 19.90, 3.2);

echo $l1->afficher() . "\n";
// PHP POO par Dupont — 29.90€
echo $l2->afficher() . "\n";
// SQL par Martin — 19.90€ [PDF: 3.2 Mo]
C#
class Livre {
    public string Titre { get; private set; }
    public string Auteur { get; private set; }
    private double prix;
    public double Prix {
        get => prix;
        set {
            if (value < 0) throw new Exception("Prix < 0");
            prix = value;
        }
    }

    public Livre(string t, string a, double p) {
        Titre = t; Auteur = a; Prix = p;
    }

    public virtual string Afficher() {
        return $"{Titre} par {Auteur} — {Prix}€";
    }
}

class LivreElectronique : Livre {
    public double TailleFichier { get; private set; }

    public LivreElectronique(string t, string a,
        double p, double taille) : base(t, a, p)
    {
        TailleFichier = taille;
    }

    public override string Afficher() {
        return base.Afficher()
             + $" [PDF: {TailleFichier} Mo]";
    }
}

// Test
Livre l1 = new Livre("C# POO", "Dupont", 29.90);
LivreElectronique l2 = new("SQL", "Martin", 19.90, 3.2);

Console.WriteLine(l1.Afficher());
Console.WriteLine(l2.Afficher());
Ce qui rapporte les points :
• extends / : → 1 pt
• parent::__construct() / base() → 1 pt
• Attribut propre + getter → 1 pt
• Surcharge afficher() avec appel parent → 2 pts
En C#, n'oublie pas virtual dans le parent et override dans l'enfant !
