Zone d'Entraînement

1. Mise en Majuscule

Affichez tous les noms de pays en majuscules.

SELECT UPPER(country) FROM Customers;

2. Initiales

Extrayez la première lettre du nom (LEFT ou SUBSTRING).

SELECT LEFT(name, 1) FROM Users;
-- Ou SUBSTRING(name, 1, 1)

3. Longueur

Listez les produits dont le nom fait plus de 10 caractères (LEN/LENGTH).

SELECT * FROM Products WHERE LENGTH(name) > 10;

4. Recherche Floue

Trouvez les utilisateurs dont le nom contient "son" (LIKE).

SELECT * FROM Users WHERE name LIKE '%son%';

5. Formatage Téléphone

Remplacez les tirets '-' par des espaces ' ' dans les numéros (REPLACE).

SELECT REPLACE(phone, '-', ' ') FROM Users;
